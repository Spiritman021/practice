Third Normal Form
3NF in DBMS

Third Normal Form removes transitive dependency. A non-prime attribute should not determine another non-prime attribute.

Must be in 2NFNo Transitive Dependency Prime AttributeCandidate Key Test
1

Rules of Third Normal Form

According to E. F. Codd, a relation is in Third Normal Form when it satisfies two rules.

Rule 1
The relation must already be in Second Normal Form.
Rule 2
The relation must not contain transitive dependency.
Theory memory: transitive dependency is the problematic case where a non-prime attribute determines another non-prime attribute.
2

Transitive Dependency

If a candidate key determines a non-prime attribute, and that non-prime attribute determines another non-prime attribute, then the second dependency is transitive.

3

Simple FD Example

Relation R(A, B, C, D) has functional dependencies:

AB → CC → D
ItemResultMeaning
Candidate keyABAB+ gives A, B, C, D.
Prime attributesA, BUsed in candidate key.
Non-prime attributesC, DNot used in candidate key.
ProblemC → DC is non-prime and D is non-prime, so transitive dependency exists.
Because of AB → C and C → D, we can derive AB → D through C. That intermediate non-prime attribute creates the 3NF violation.
4

Direct 3NF Checking Condition

For every functional dependency X → Y, the table is in 3NF if at least one condition is true.

Condition 1
Left-hand side X is a candidate key or super key.
Condition 2
Right-hand side Y is a prime attribute.
Notice the condition uses OR. If either side of the test passes, that FD is acceptable for 3NF.
5

Worked Question: AB → CD, D → A

Relation R(A, B, C, D) has functional dependencies:

AB → CDD → A
Find Candidate Keys
StepResult
Right-side attributesC, D, A appear on RHS, so B must be in every candidate key.
AB+AB gives C and D, so AB determines all attributes. Candidate key = AB.
Replacement checkA appears on RHS of D → A, so replace A with D and test DB.
DB+DB gives A from D, then AB gives C and D. Candidate key = DB.
Candidate keys are AB and DB. Prime attributes are A, B, D. Non-prime attribute is C.
