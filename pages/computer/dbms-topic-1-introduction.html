04

What is a DBMS?

📖
Definition: A Database Management System (DBMS) is a software platform that allows users to easily access data and perform operations on it — regardless of where the data is physically stored.

Core Operations (DML operations):

➕
Insertion
Adding new records/data into the database.
e.g., new student enrollment.
✏️
Updation
Modifying existing data.
e.g., changing a student's address.
🗑️
Deletion
Removing unwanted records.
e.g., deleting a cancelled booking.
📍
Location Independence: The user doesn't need to know where data is physically stored. A user in Delhi can book a ticket stored on an IRCTC server in Mumbai — seamlessly.

Popular DBMS Software:

Microsoft SQL Server
Oracle Database
MySQL
IBM DB2

MariaDB
05

The Concept of RDBMS

🔑
RDBMS stands for Relational Database Management System — a DBMS designed specifically to manage structured data stored in Relations (Tables).

How it all connects:

Structured Data Organized in rows & columns
→
Stored as Relations Tables (like Excel sheets)
→
Managed by RDBMS Insert, Update, Delete
→
= RDBMS Focus Core of GATE syllabus
  • The B.Tech, GATE, and UGC NET syllabus focuses exclusively on RDBMS — structured data and relational tables are the primary concern.
  • A Relation = a Table. Rows represent records; columns represent attributes. Think of a well-organized Excel spreadsheet.
  • RDBMS supports all standard DML operations: INSERT, UPDATE, and DELETE on tabular data.
✦

Quick Recap

  • Database = Collection of related data (not random — purposefully grouped).
  • DBMS = Software to access and operate on that data (insert, update, delete).
  • Structured Data is stored as Relations (Tables) — this is what RDBMS manages.
  • 90% of the world's data is unstructured; managed via Big Data / Hadoop.
  • RDBMS = DBMS specifically for structured data in relational (table) format.
  • Key DBMS software: MySQL, Oracle, MS SQL Server, IBM DB2, PostgreSQL.

Q&A Flashcards

+
Q1What is a Database?
A collection of related data. The data is purposefully grouped — not random. Example: IRCTC stores train & passenger data, not passport data.
+
Q2What is a DBMS?
A Database Management System — software that lets users access and operate on data. Core operations: Insert, Update, Delete. Users don't need to know where data is stored physically.
+
Q3What are the 3 types of data?
1. Structured — stored in tables (rows & columns); managed by RDBMS.
2. Semi-Structured — tag/marker based (JSON, XML); managed by Document DBs.
3. Unstructured — no fixed format (photos, videos, web pages); managed by Big Data / Hadoop.
+
Q4What percentage of world data is unstructured?
90% of the data on Earth is unstructured. Examples: photos, videos, web pages. Managed using Big Data and Hadoop ecosystems.
+
Q5What is RDBMS and how does it differ from DBMS?
RDBMS = Relational DBMS. It is a DBMS specifically designed to manage structured data stored as Relations (Tables). All RDBMS are DBMS, but not all DBMS are RDBMS.
+
Q6What does "Relation" mean in RDBMS?
A Relation = Table. It has rows (records/tuples) and columns (attributes/fields). Similar to an Excel sheet. IRCTC search results showing train number, seats, departure time is a relation.
+
Q7Name 4 popular DBMS software platforms.
MySQL (open-source, widely used), Oracle Database (enterprise), Microsoft SQL Server, IBM DB2, and also PostgreSQL, MariaDB.
+
Q8What is a Database System?
A Database System = Database + DBMS. The Database holds the data; the DBMS is the software layer that manages access and operations on that data.
