"A number of" vs "The number of"

A number of = a lot of (many) → plural verb · The number of = the count → singular verb · Same rule for "A variety of" / "The variety of"

→ The Core Rule
★ From the book (Rule 51): Rule for using "A number of" and "The number of".
We use the following structures with these two:
  • "A number of" + Plural Noun + Plural Verb
  • "The number of" + Plural Noun + Singular Verb
Similar rule is there for "A variety of" and "The variety of".
A number of
= a lot of / बहुत सारे
Structure:
A number of + Plural Noun + Plural Verb

"A number of students are waiting here."
"A number of tickets were sold."

Because we're talking about many → plural verb.
The number of
= the count / संख्या
Structure:
The number of + Plural Noun + Singular Verb

"The number of students is fifty."
"The number of employees is increasing."

Because the count itself is one → singular verb.
A number of → Plural Noun + Plural Verb
The number of → Plural Noun + Singular Verb
💡
Memory Trick:
A number of = "A lot of" → many things → plural verb (are, were, have…)
The number of = "The count" → the count is ONE thing → singular verb (is, was, has…)
★ All Examples from Book
A number of — plural verb
✓
A number of students are waiting here. [Correct]
"A number of" = a lot of → many students → plural verb "are". ✓
✓
A number of tickets were sold almost immediately. [Correct]
"A number of" = many tickets → plural verb "were". ✓
The number of — singular verb
✓
The number of students is fifty. [Correct]
"The number of" = the count is 50 → singular verb "is". ✓
✓
The number of employees taking legal action over work-related disease is increasing. [Correct]
"The number of" = the count → singular verb "is". ✓
📌
Also applicable to:
A variety of + Plural Noun + Plural Verb
The variety of + Plural Noun + Singular Verb
The same Article (a/the) determines the verb — same logic applies.

📋 Rule 51 — Quick Recap

  • A number of (= a lot of) → Plural Noun + Plural Verb: "A number of students are waiting." ✓
  • The number of (= the count) → Plural Noun + Singular Verb: "The number of students is 50." ✓
  • Noun is ALWAYS plural in both cases — only the verb changes.
  • Same rule: "A variety of" → plural verb · "The variety of" → singular verb

"It is time" / "It is high time" / "It is about time"

No subject → to + V1 · With subject → Subject + V2 (past form). Means: अब बहुत समय हो गया है / now is the moment

→ The Core Rule — Two Conditions
★ From the book (Rule 52): Use of the phrase "It is time", "It is high time" or "It is about time".
With these phrases, we follow either of the two conditions:
(1) The phrase should be followed by "to + V1" if there is no Subject attached to it.
(2) If the phrase is directly followed by a Subject, we use the verb form "V2" (past tense) after the subject.

*NOTE: We use the expression "it's time + subject + past verb form" to refer to the present moment.
Condition 1 — No Subject
When NO subject follows the phrase:

Formula: It is time + to + V1

"It is time to work hard."
"It is time to ask for a raise."
"It is high time to realize mistakes."
Condition 2 — With Subject
When a subject (I/you/he/she/we/they) follows:

Formula: It is time + Subject + V2

"It is time he worked hard."
"It is time we went home."
"It is high time she realized her mistakes."
It is (high) time + to + V1 [no subject]  OR  It is (high) time + Subject + V2 [with subject]
💡
Why V2 (past form) with subject?
"It is high time" refers to the PRESENT moment but uses past tense (V2) after the subject. This is a special grammatical structure in English — like subjunctive — to indicate urgency or that something should have been done by now.

"It is high time he worked" = He should work NOW (present urgency expressed via past form).
★ All Examples from Book & Class
No subject → to + V1
✓
It is time to work hard. [Correct]
No subject after "It is time" → to + V1 ("to work"). ✓
✓
It is time to ask for a raise. [Correct]
No subject → to + V1 ("to ask"). ✓
✓
It is high time to realize mistakes. [Correct]
No subject → to + V1 ("to realize"). ✓
With subject → V2 (past form)
✓
It is time he worked hard. [Correct — "he" is subject]
Subject "he" present → V2 "worked". ✓
✓
Gosh! It's almost midnight. It is time we went home. [Correct — "we" is subject]
Subject "we" present → V2 "went". ✓
✓
It is high time she realized her mistakes. [Correct — "she" is subject]
Subject "she" present → V2 "realized". ✓
✓
It is high time you worked hard. [Correct]
Subject "you" → V2 "worked". ✓

📋 Rule 52 — Quick Recap

  • Phrase: "It is time" / "It is high time" / "It is about time" = बहुत समय हो गया / now is the moment
  • No subject after phrase → to + V1: "It is time to work." ✓
  • Subject present → Subject + V2: "It is time he worked." ✓ | "It is time we went home." ✓
  • V2 (past form) is used even though meaning is present — special English structure (subjunctive-like)
  • Subjects: I/you/he/she/it/we/they → followed by V2 always in this structure

"Angry with" vs "Angry at" · Same Rule for "Annoy"

angry with + person/living thing · angry at + non-living thing/behavior · Identical rule for "annoy/annoyed"

→ The Core Rule
★ From the book (Rule 53): Use of preposition with the word "Angry":
(1) We use the preposition "with" with "angry" whenever we refer a person or living thing for the purpose.
(2) We use the preposition "at" with "angry" whenever we refer something other than person or living thing.
*NOTE: We follow the same rule with the word "Annoy".
angry with+ Person / Living thing
Use "with" when you are angry at a person or living thing (human, animal etc.)

✓ "I got really angry with her."
✓ "I am angry with Shyam because he did not go to the class."
✓ "She was angry with him."
angry at+ Thing / Behavior
Use "at" when you are angry about a non-living thing, situation, or behavior.

✓ "The local residents were angry at the lack of parking spaces."
✓ "They are angry at the way they have been treated."
✓ "I am angry at his behavior."
angry with [person/living]  ·  angry at [thing/behavior/situation]
★ All Examples from Book & Class
angry WITH — person / living thing
✓
I got really angry with her. [Correct]
"her" = a person → angry with. ✓
✓
I am angry with Shyam because he did not go to the class. [Correct]
"Shyam" = a person → angry with. ✓
✗
I am angry at Ram. [Incorrect]
"Ram" is a person → must use "angry with". ✗
angry AT — thing / behavior / non-living
✓
The local residents were angry at the lack of parking spaces. [Correct]
"lack of parking spaces" = non-living situation → angry at. ✓
✓
They are angry at the way they have been treated. [Correct]
"the way they have been treated" = behavior/situation → angry at. ✓
✓
I am angry at his behavior. [Correct]
"his behavior" = not a person, it's behavior/thing → angry at. ✓
✗
They are angry with the lack of parking spaces. [Incorrect]
"lack of parking spaces" = non-living → must use "at". ✗
→ Same Rule for "Annoy / Annoyed"
★ From the book: *NOTE: We follow the same rule with the word "Annoy".
Annoy = परेशान होना / irritated होना / गुस्सा आना
annoyed WITH + person
✓ "You are annoyed with someone."
✓ "She was annoyed with him."
✓ "I am annoyed with my friend."
annoyed AT + thing
✓ "You are annoyed at his behavior."
✓ "She is annoyed at the lack of resources."
✓ "I am annoyed at the delay."
💡
Quick memory trick for BOTH angry & annoyed:
→ Living / Person → use WITH (you're directly with that person)
→ Non-living / Behavior / Situation → use AT (you're pointing at the thing/issue)

📋 Rule 53 — Quick Recap

  • angry with + person/living thing: "I am angry with Ram." ✓ | "angry at Ram" ✗
  • angry at + thing/behavior/situation: "angry at the lack of parking" ✓ | "angry at his behavior" ✓
  • Same rule for annoy/annoyed: "annoyed with someone" ✓ | "annoyed at something" ✓
  • Trick: WITH = person (you're WITH them) · AT = thing (you're pointing AT it)
