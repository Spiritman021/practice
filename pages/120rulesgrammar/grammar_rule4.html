Golden Rules of Conditional Sentences

3 Types of Conditionals + Zero Conditional (Universal Truth) exception

→ What is a Conditional Sentence?

A Conditional Sentence is a sentence which is formed of two parts:

PartNameWhat it containsStarts with
Part 1 If-Clause (Condition) The condition that needs to be met "If…"
Part 2 Main Clause (Result) The result/consequence if the condition is met The outcome
Example: "If I reach home late, my mother will welcome me with a big stick."
→ Condition = "If I reach home late"
→ Result = "my mother will welcome me with a big stick"

There are three types of conditional sentences, each based on how realistic or possible the condition is.

TypeCondition
Type ICondition possible to fulfil (real/likely)
Type IICondition in theory possible to fulfil (hypothetical/unlikely)
Type IIICondition not possible to fulfil — too late (impossible/past)

Quick memory formula:
Type I → Simple Present → will + V1
Type II → Simple Past → would + V1
Type III → Past Perfect → would + have + V3
1 Type I — Condition Possible to Fulfil (Real / Likely)

If-clause: Simple Present  |  Main clause: will / shall / can / may / must + V1
Used when the condition is real and likely to happen.

Examples
✓
If I study, I will pass the exam.
If-clause: Simple Present ("study") → Main: will + V1 ("will pass"). [Type I]
✓
If your files are in order, you may leave at once.
If-clause: Simple Present ("are") → Main: may + V1 ("may leave"). [Type I]
✓
If you want to lose weight, you must eat less fat diet.
If-clause: Simple Present ("want") → Main: must + V1 ("must eat"). [Type I]
Note: In Type I, the main clause can use any present modal: will, shall, can, may, must + V1. Not just "will" — all are correct depending on the meaning intended.
2 Type II — Condition in Theory Possible (Hypothetical / Unlikely)

If-clause: Simple Past  |  Main clause: would / could / might / should + V1
Used when the condition is hypothetical, imaginary, or unlikely — possible in theory but not happening in reality right now.

Examples
✓
If I studied, I would pass the exam.
If-clause: Simple Past ("studied") → Main: would + V1 ("would pass"). [Type II]
✓
If I lived near my metro station, I would be in time for work.
If-clause: Simple Past ("lived") → Main: would + V1 ("would be"). [Type II]
Note: "be" is the V1 of "is/am/are"
✓
If I were you, I would plant some saplings around the house.
If-clause: Simple Past ("were") → Main: would + V1 ("would plant"). [Type II]
Note: In Type II, "were" is used for all persons — even "I" → "If I were you…" (not "was")
Special Note about "were": In Type II conditionals, we use "were" for all subjects (I, he, she, it, we, you, they) — not "was". So it's always "If I were you…" not "If I was you…" — even though grammatically "was" would seem correct.
3 Type III — Condition Not Possible to Fulfil (Too Late / Past)

If-clause: Past Perfect (had + V3)  |  Main clause: would / could / might / should + have + V3
Used when the condition cannot be fulfilled — the opportunity has already passed. We are imagining a different past.

Examples
✓
If I had studied, I would have passed the exam.
If-clause: Past Perfect ("had studied") → Main: would + have + V3 ("would have passed"). [Type III]
✓
If we had found him earlier, we could have arrested him and saved the crime.
If-clause: Past Perfect ("had found") → Main: could + have + V3 ("could have arrested"). [Type III]
✓
Had you obeyed the orders, this fire accident would have been averted.
If-clause: Past Perfect starting with "Had" (inverted form, no "if") → Main: would + have + V3. [Type III]
Note: "Had + subject + V3…" is an inverted conditional — it means "If you had obeyed…". This is ALSO a conditional sentence even though it doesn't start with "If".
Important: Type III sentences can also start with "Had + subject + V3…" — this is the inverted form of a Type III conditional. The word "If" is dropped and "Had" comes first. The meaning is exactly the same.

"Had you obeyed the orders…" = "If you had obeyed the orders…"

⚠ Exception — Zero Conditional (Universal Truth)

There is one important exception where the three Golden Rules of Rule 4 do NOT apply — this is called the Zero Conditional.

What is a Universal Truth?
A universal truth is something that is always true — regardless of when or where. It is a scientific fact or a general fact of nature.

Key point you already know: Any Universal Truth is always written in Simple Present Tense.

Examples of universal truths: "The sun rises in the east." / "Honey is sweet." / "Water boils at 100°C."

The Rule for Zero Conditional:
When a conditional sentence expresses a universal truth, BOTH the if-clause AND the main clause use Simple Present Tense. The three Golden Rules (Type I, II, III) do NOT apply.

If-ClauseMain Clause
Normal Rules (Type I/II/III) Varies (Simple Present / Past / Past Perfect) Varies (will/would/would have + V)
Zero Conditional (Universal Truth) Simple Present Simple Present
Examples — Zero Conditional
✓
If water reaches 100-degree Celsius, it boils.
Universal truth → Both clauses in Simple Present. [Universal truth]
✓
If you heat water, it turns into vapour.
Universal truth → Both clauses in Simple Present. [Universal truth]
★ NOTE from the book: The given rule (Type I / II / III) is not applicable in the case of universal truth. In such sentences, always use Simple Present Tense in both clauses — never "will", "would", or "would have".

📋 Rule 4 — Quick Recap

  • Type I (Possible) → If-clause: Simple Present → Main: will/shall/can/may/must + V1
  • Type II (Hypothetical) → If-clause: Simple Past → Main: would/could/might/should + V1
  • Type III (Impossible/Too Late) → If-clause: Past Perfect (had + V3) → Main: would/could/might/should + have + V3
  • Type III can also appear as an inverted form: "Had + subject + V3…" (no "If") — same meaning
  • Zero Conditional (Exception): Universal Truths → Both clauses in Simple Present. Type I/II/III rules do NOT apply.
  • In Type II, always use "were" for all subjects (even "I") — never "was"
