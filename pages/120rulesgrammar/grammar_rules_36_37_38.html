Comparison Between Two Qualities of One Object

When comparing two qualities of the SAME thing → always use "more" + Positive Degree · Never use Comparative Degree (-er / more separately)

→ The Core Rule
★ From the book (Rule 36): Comparison between two qualities of an object.
Whenever two qualities of an object are compared, we always use the word "more". We don't use comparative degree in that case.

When you compare two qualities of the same person or object, you cannot use the comparative degree (taller, smarter, redder). You must use "more" + positive degree of both adjectives.

✗ WRONG — Comparative Degree
Ram is taller than smarter.
This pen is redder than white.
This book is heavier than larger.
✓ CORRECT — more + Positive Degree
Ram is more tall than smart.
This pen is more red than white.
This book is more heavy than large.
[Subject] is more [Positive Adj 1] than [Positive Adj 2].
Never: [Comparative Adj 1] than [Comparative Adj 2]
★ All Examples from Book & Class
✗
This pen is redder than white. [Incorrect]
Two qualities of one pen compared → cannot use comparative "redder". ✗
✓
This shirt is more white than red. [Correct]
"more" + positive degree "white" and "red". ✓
✗
This book is heavier than larger. [Incorrect]
Two qualities of one book → comparative degree not allowed. ✗
✓
This book is more heavy than large. [Correct]
"more" + positive degree. ✓
✓
This umbrella is more beautiful than strong. [Correct]
Two qualities of umbrella (beauty vs strength) → "more" + positive degree. ✓
✓
Ram is more tall than smart. [Correct]
Two qualities of Ram (tall vs smart) → "more" + positive degree. ✓
💡
How to identify this situation:
Ask: Are both adjectives describing the same subject?
→ Yes (comparing two qualities of Ram / this book / this pen) → use "more" + Positive Degree for BOTH adjectives.
→ No (Ram compared to Shyam) → normal comparative ("taller than Shyam") is fine.

📋 Rule 36 — Quick Recap

  • Two qualities of the same thing compared → always use "more" + Positive Degree
  • NEVER use Comparative Degree (-er / more…er) when both adjectives describe the same subject
  • "This pen is more red than white" ✓ | "This pen is redder than white" ✗
  • "Ram is more tall than smart" ✓ | "Ram is taller than smarter" ✗
  • "This umbrella is more beautiful than strong" ✓
37

"more than one" / "many a" → Singular  |  "many" / "a great many" / "a good many" → Plural

Structure rule: noun and verb both follow the nearest quantifier word

→ The Core Rule
★ From the book (Rule 37):
Whenever "more than one" or "many a" is used to quantify a noun, it is always followed by a singular noun and a singular verb.

And whenever "Many", "A good many", "A great many" are used to quantify a noun, they are always followed by a plural noun and a plural verb.
→ SINGULAR Noun + SINGULAR Verb
More than one + singular noun + singular verb
Many a + singular noun + singular verb
One + singular noun + singular verb

Why? "one" and "a" both mean one (ek) → so the nearest noun-word is singular → singular verb follows.
→ PLURAL Noun + PLURAL Verb
Many + plural noun + plural verb
A great many + plural noun + plural verb
A good many + plural noun + plural verb

Why? "many" = बहुत सारे → clearly plural → plural noun and verb follow.
Verb agrees with the nearest word:
more than one → nearest = "one" → Singular  |  many a → nearest = "a" → Singular
many → Plural  |  a great many → Plural
→ Complete Structure Table — from Book
QuantifierNounVerbExample

OneSingularSingular (is/was)One student is absent.

"many a" → nearest word "a" (= one) → verb "is". ✓
a great many / a good many → Plural

"a great many" → plural → verb "are". ✓
✓

"a great many" → plural → "are". ✓
many → Plural
✓
Many students are absent today. [Correct — Plural verb]
"many" → plural → verb "are". ✓
✓
Many tigers are missing from the national park. [Correct]
"many" → plural → "are". ✓
one → Singular (for reference)
✓
One student is absent today. [Correct — Singular verb]
"one" → singular → "is". ✓
💡
The One-Line Memory Trick:
If you see "one" or "a" in the quantifier → Singular noun + Singular verb.
(more than one · many a · one)

If you see "many" standing alone or in "a great/good many" → Plural noun + Plural verb.
(many · a great many · a good many)

✓

📋 Rule 37 — Quick Recap

  • more than one + singular noun + singular verb → "More than one student is" ✓
  • many a + singular noun + singular verb → "Many a student is" ✓
  • many / a great many / a good many + plural noun + plural verb → "Many students are" ✓
  • Key: "one" and "a" → singular | "many" (alone) → plural
  • "More than one student are" ✗ | "Many a students are" ✗ — both wrong
38

"doubt" / "doubtful" → if / whether    |    "no doubt" / "not doubtful" → that

Examiner's favourite · Connector depends on whether doubt is present or absent

→ The Core Rule
★ From the book (Rule 38):
Whenever we get the word "doubt" or "doubtful" in a sentence, we use the connector either "if" or "whether".

However, in case we get the words "no doubt" and "not doubtful", we use the connector "that".
Word in sentenceConnector to useReason
doubt / doubtful → if  /  whether Uncertainty exists → answer is yes or no → use if/whether
no doubt / not doubtful → that No uncertainty → definite statement → use "that"
💡
Simple Logic:
Doubt present = shak hai → answer could be YES or NO → if / whether
No doubt = shak nahi → it's a sure statement → that
★ All Examples from Book & Class
"no doubt" → connector must be "that" (NOT if/whether)
✗
I have no doubt whether it will rain or not. [Incorrect]
"no doubt" → connector must be "that", not "whether". ✗
✓
I have no doubt that it will rain. [Correct]
"no doubt" → "that". ✓
✗
He has no doubt if Shyam is his best friend. [Incorrect]
"no doubt" → must use "that", not "if". ✗
✓
He has no doubt that Shyam is his best friend. [Correct]
"no doubt" → "that". ✓
"doubt" (present) → connector must be "if" or "whether" (NOT that)
✗
There is a doubt that he goes there or not. [Incorrect]
"doubt" is present → connector must be "if" or "whether", not "that". ✗
✓
There is a doubt whether he goes there or not. [Correct]
"doubt" present → "whether". ✓
More examples — class
✓
I have a doubt whether / if he is honest. [Correct]
"a doubt" = shak hai → if/whether. Both correct. ✓
✗
I have a doubt that he is not honest. [Incorrect]
"a doubt" is present → "that" cannot be used. Must use "if/whether". ✗
✓
I have no doubt that it will rain. [Correct]
"no doubt" = no uncertainty → "that". ✓
✓
He has no doubt that Shyam is his best friend. [Correct]
"no doubt" → "that". ✓
⚠️
Common Mistake — "or not" with "no doubt":
"I have no doubt that it will rain or not." → The "or not" is also wrong when "no doubt" is used, because there's no uncertainty. Remove "or not" in sentences with "no doubt".

✏️ Practice — Correct or Incorrect?

✗

"no doubt" → must be "that", not "whether" ✗
✓

"no doubt" → "that" ✓
✗

"a doubt" present → must be "if/whether" ✗
✓
There is a doubt whether she passed.
"a doubt" → "whether" ✓

📋 Rule 38 — Quick Recap

  • doubt / doubtful in sentence → use connector "if" or "whether"
  • no doubt / not doubtful → use connector "that"
  • "I have no doubt that it will rain" ✓ | "I have no doubt whether it will rain" ✗
  • "There is a doubt whether he goes there" ✓ | "There is a doubt that he goes there" ✗
  • Logic: Doubt = uncertainty = yes/no = if/whether | No doubt = certainty = that

Rule 37 — more than one / many a vs many / a great many Quantifiers
Singular (one/a):More than one student is · Many a student is · One student is — singular noun + singular verb
Plural (many):Many students are · A great many students are · A good many students are — plural noun + plural verb
Trick:See "one" or "a" in quantifier → singular | See "many" alone → plural
