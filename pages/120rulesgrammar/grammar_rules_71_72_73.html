71

Negative Words in Statement → Always POSITIVE Question Tag

seldom · barely · hardly · scarcely · rarely · never · few · little · forbid/forbade → all these words make the statement negative → so the question tag must be positive

→ The Core Rule
★ Rule 71: Certain words are inherently negative in meaning, even though the sentence looks positive (no "not" written). Since the sentence is already negative in meaning → the question tag must be POSITIVE.

Remember Rule 69: Question tag is always OPPOSITE of the statement. These sentences are NEGATIVE in meaning → so the tag is POSITIVE.
Statement with negative word = Negative statement → POSITIVE question tag
Even if there is no "not" in the sentence — these words already make it negative!
Inherently Negative Words (memorise all)
seldom (ना के बराबर) barely (ना के बराबर) hardly (ना के बराबर) scarcely (ना के बराबर) rarely (ना के बराबर) never (कभी नहीं) few (कम) little (थोड़ा) forbid / forbade
★ All Examples from Class
StatementNegative Word✓ Tag (Positive)
I seldom go to his house,seldomdo I?
He barely comes to my house,barelydoes he?
They barely have enough money to pay rent,barelyhave they?
I never realized you knew my brother,neverdid I?
We hardly go to concerts,hardlydo we?
I seldom drive my car into the city,seldomdo I?
I scarcely acknowledged the fact that he was there,scarcelydid I?
I forbade him to go there,forbadedid I?
💡
Quick Trick:
Does the sentence LOOK positive but FEEL negative? Look for these hidden negative words.
"I seldom go" = I (basically) don't go → negative in meaning → positive tag.
If you spot any of these 9 words → the tag is ALWAYS positive!

📋 Rule 71 — Quick Recap

  • Negative words: seldom · barely · hardly · scarcely · rarely · never · few · little · forbid/forbade
  • These words make the statement negative in meaning → question tag is always POSITIVE
  • "I seldom go to his house, do I?" ✓ · "He barely comes, does he?" ✓ · "I never realized, did I?" ✓
72

Question Tag Tense = Always SAME as Statement Tense

The helping verb in the question tag must ALWAYS match the tense of the main statement — they can never be different

→ The Core Rule
★ Rule 72: The tense of the question tag is always the same as the tense of the main statement. The helping verb used in the statement is the same helping verb used in the question tag. They can never be different tenses.
Statement tense = Question tag tense  ·  Same helping verb in BOTH
Simple Past → "did/didn't" · Past Perfect → "had/hadn't" · Present Continuous → "am/aren't" etc.
★ All Examples — Tense Matching
StatementTenseCorrect Tag ✓Wrong Tag ✗
I forbade him to go there, Simple Past did I? don't I? ✗
She had done it, Past Perfect hadn't she? didn't she? ✗
I am doing the work which you gave me, Present Continuous aren't I? don't I? ✗ / amn't I? ✗
She completed the work on time, Simple Past didn't she? hasn't she? ✗
They have been working hard, Present Perfect Continuous haven't they? didn't they? ✗
He will come tomorrow, Simple Future won't he? doesn't he? ✗
✓
I am doing the work, aren't I? [Correct — Present Continuous → "am" → "aren't I"]
"am" → tag uses "aren't I" (remember: "amn't I" doesn't exist). Tense = Present Continuous. ✓
✗
I am doing the work, don't I? [Incorrect]
"don't" = Simple Present tense, but statement is in Present Continuous → tense mismatch. ✗
💡
Formation steps for tense-matching:
1. Identify the tense of the statement (look at the helping verb).
2. Use the SAME helping verb in the tag.
3. Make it opposite (positive→negative / negative→positive).
4. Use the appropriate pronoun.

"She had done it" → tense = Past Perfect → tag uses "had" → "hadn't she?" ✓

📋 Rule 72 — Quick Recap

  • Question tag tense = always same as statement tense (same helping verb)
  • Simple Past → "did/didn't": "She completed the work, didn't she?" ✓
  • Past Perfect → "had/hadn't": "She had done it, hadn't she?" ✓
  • Present Continuous → "am/aren't": "I am working, aren't I?" ✓ (never "don't I?" ✗)
  • Tenses can NEVER be different — "She completed... hasn't she?" ✗ (tense mismatch)
73

"each" and "every" — PLURAL in Question Tags (opposite of statements!)

In normal statements: each/every → singular verb. In QUESTION TAGS: each/every → always treated as PLURAL → "they/them" → "aren't they?", "haven't they?", "won't they?"

→ The Core Rule
★ Rule 73: In STATEMENTS — "each" and "every" take a singular verb (we studied this before in Rule 7). BUT in QUESTION TAGS — "each" and "every" are treated as PLURAL. So the question tag uses "they" and a plural auxiliary. This is an exception/reversal specifically for question tags.
In STATEMENTS (Rule 7)
each/every → singular verb
"Each student is talented." ✓
"Every girl is smart." ✓
In QUESTION TAGS (Rule 73)
each/every → PLURAL → "they"
"…aren't they?" ✓ (not "isn't he/she?")
"…won't they?" ✓ (not "won't he?")
each/every in question tags → always PLURAL → use "they"
"aren't they?" · "haven't they?" · "won't they?" · "didn't they?" — NEVER "isn't she/he?" or "hasn't he?"
★ All Examples from Class
Statement✓ Correct Tag✗ Wrong Tag
Each student is talented, aren't they? isn't he/she? ✗
Every girl is smart, aren't they? isn't she? ✗
Each one has done it, haven't they? hasn't he? ✗
Everyone will die, won't they? won't he? ✗
Every student has passed, haven't they? hasn't he? ✗
✓
Each student is talented, aren't they? [Correct]
"each" in question tag → plural → "they" → "aren't they?" ✓
✗
Each student is talented, isn't he/she? [Incorrect]
In question tags, "each/every" → plural → must use "they", not "he/she". ✗
✓
Everyone will die, won't they? [Correct]
"everyone/every" in question tag → plural → "won't they?" ✓ (Note: "won't" = "will not" short form)
⭐
Key Exception to Remember:
This is the REVERSE of the normal rule.
In statements: each/every → singular (e.g., "Each student IS…")
In question tags: each/every → PLURAL (e.g., "…aren't THEY?")
This is one of the most common exam mistakes in question tags!

📋 Rule 73 — Quick Recap

  • Statements: each/every → singular verb: "Each student is talented." ✓
  • Question tags: each/every → PLURAL → pronoun = "they": "Each student is talented, aren't they?" ✓
  • NEVER use "isn't she?" / "isn't he?" / "hasn't he?" in tags with each/every → always "aren't they?" / "haven't they?" / "won't they?" ✓
  • "Everyone will die, won't they?" ✓ · "Every girl is smart, aren't they?" ✓
