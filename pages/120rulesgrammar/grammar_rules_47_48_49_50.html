47

"Quite" and "All" — Never Used Together · Quite vs Quiet

"quite all" is always wrong · Quiet = noun/adjective (silent) · Quite = adverb (completely)

→ Part A: Quite vs Quiet — Don't Confuse These Two!
★ From the book (Rule 47 NOTE): Do not get confused with the similar sounding words like "Quiet" and "Quite".
Quiet is a Noun which means the state of being silent.
Quiet
Noun / Adjective
Spelling: Q-U-I-E-T
Meaning: शांत / silent / the state of being silent

"He is a quiet man." (शांत इंसान)
"Keep quiet." (शांत रहो)
"She remains quiet all day."
"A teacher asks us to keep quiet."
Quite
Adverb
Spelling: Q-U-I-T-E
Meaning: completely / काफी / अच्छा खासा / पूरी तरह

"This is quite expensive." (काफी महंगा है)
"He is quite rich." (वह काफी अमीर है)
"I am quite sure."
(Used as degree — modifies adjectives)
💡
Quick trick:
Quiet has an E before the T → "Quiet" → think of it ending in quiet silence.
Quite has the E at the end → "Quite" → think of "completely done".
→ Part B: "Quite" and "All" — NEVER Together (Superfluous)
★ From the book (Rule 47): Words like "Quite" and "all" are never used together in a sentence.
"quite" means "completely" and "all" also implies completeness → using both = superfluous error.
"quite all" is ALWAYS wrong · Use either "quite" OR "all" — never both together
Examples from Book
✗
He is quite all well. [Incorrect]
"quite" + "all" together = superfluous. ✗
✓
He is quite well. [Correct]
Use only "quite". ✓
✗
He has done quite all the work. [Incorrect]
"quite" + "all" = superfluous. ✗
✓
He has done all the work. [Correct]
Use only "all". ✓

📋 Rule 47 — Quick Recap

  • Quiet (Q-U-I-E-T) = Noun/Adjective = silent / शांत: "Keep quiet" ✓ | "She is a quiet person" ✓
  • Quite (Q-U-I-T-E) = Adverb = completely / काफी: "This is quite expensive" ✓ | "He is quite rich" ✓
  • "quite all" = ALWAYS wrong (superfluous) → use either "quite" OR "all"
  • "He is quite well." ✓ | "He is quite all well." ✗
  • "He has done all the work." ✓ | "He has done quite all the work." ✗

"ago" → Simple Past  ·  "before" → Past Perfect

ago = one event → Simple Past · before = two events → earlier event: Past Perfect, later event: Simple Past

→ The Core Rule
★ From the book (Rule 48): Determination of tense based on words like "ago" and "before":
• We generally use Simple Past Tense when the word "ago" is mentioned in the sentence.
• In the case of "before", we use Past Perfect Tense (for the earlier action).
ago → Simple Past
Used when ONLY ONE event is given.

Formula: [Subject] + V2 + [time] + ago

"I met him five years ago."
"He went two seconds ago."
"She submitted the work five hours ago."
before → Past Perfect
Used when TWO events are given.

Earlier event: had + V3
Later event: V2 (Simple Past)

"The patient had died before the doctor came."

BEFORE

The arrow of time → Earlier (further in past) uses Past Perfect · Later uses Simple Past
★ All Examples from Book
ago examples — Simple Past
✗
He had gone two seconds ago. [Incorrect]
"ago" = one event → Simple Past needed, not Past Perfect. ✗
✓
He went two seconds ago. [Correct]
"ago" + one event → Simple Past "went". ✓
✗
She had submitted the work five hours ago. [Incorrect]
"ago" = one event → Simple Past, not Past Perfect. ✗
✓
She submitted the work five hours ago. [Correct]
"ago" + one event → Simple Past. ✓
before examples — Past Perfect for earlier event
✗
The patient died before the doctor came. [Incorrect]
Two events: "dying" came before "doctor's arrival". Earlier event needs Past Perfect "had died". ✗
✓
The patient had died before the doctor came. [Correct]
Earlier event = "had died" (Past Perfect) | Later event = "came" (Simple Past). ✓
Class Examples
✓
I had got a selection before I turned 25. [Correct]
Two events: getting selection (earlier = Past Perfect "had got") | turning 25 (later = Simple Past "turned"). ✓
✓
She had finished the work before her mother gave her a thrashing. [Correct]
Earlier: "had finished" (Past Perfect) | Later: "gave" (Simple Past). ✓
✓
I met him five years ago. [Correct]
One event only → "ago" → Simple Past "met". ✓

📋 Rule 48 — Quick Recap

  • ago = ONE event in past → Simple Past (V2): "He went two seconds ago." ✓
  • before = TWO events in past → Earlier event: Past Perfect (had + V3) | Later event: Simple Past (V2)
  • "She had submitted the work five hours ago." ✗ → "She submitted…" ✓
  • "Patient had died before the doctor came." ✓ (two events)
  • "I had finished before she arrived." ✓ | "I finished before she arrived." ✗

Common Possession vs Split Possession — Use of Possessive Sign

Common: apostrophe only at last noun · Split: apostrophe with each noun → must use plural noun

→ The Two Types of Possession
★ From the book (Rule 49): Use of Possessive sign?
(1) Common Possession: When two nouns are closely connected or implying the common possession, the possessive sign is put to the latter (last one).
(2) Split Possession: Each of two or more connected nouns implying separate possession must take the possessive sign.
✓ Common Possession
Both own the SAME thing(s) jointly.
→ Apostrophe only at the last noun.

Ram and Shyam's car
(one car, both own it)

Ram and Shyam's cars
(many cars, both own all)
✗/✓ Split Possession
Each has their OWN separate thing(s).
→ Apostrophe with each noun.
→ MUST use plural noun (never singular).

Ram's and Shyam's cars
(Ram has his own, Shyam has his own)

⚠ Singular noun NEVER allowed in split.
Common → apostrophe at END only · Split → apostrophe on EACH · Split → PLURAL noun always
★ All Examples from Book
Common Possession — apostrophe only at last noun
✓
Ram and Shyam's car is being decorated for Balram's marriage. [Correct]
Common possession: one car jointly owned by both → apostrophe at "Shyam's" only → singular "car" → singular verb "is". ✓
✓
Ram and Shyam's cars are being decorated for Balram's marriage. [Correct]
Common possession: many cars jointly owned by both → apostrophe at "Shyam's" only → plural "cars" → plural verb "are". ✓
Split Possession — apostrophe with EACH noun + PLURAL noun mandatory
✗
Ram's and Shyam's car is being decorated for Balram's marriage. [Incorrect]
Split possession = each has their own car → at least 2 cars → singular "car" is WRONG. ✗
✓
Ram's and Shyam's cars are being decorated for Balram's marriage. [Correct]
Split possession: Ram's own car + Shyam's own car → plural "cars" → plural verb "are". ✓
⚠️
Split Possession can NEVER take a Singular Noun!
If possession is split (each person has separate things), there must be at least 2 things → the noun MUST be plural.
"Ram's and Shyam's car" ✗ (impossible — if separate, must be "cars") · "Ram's and Shyam's cars" ✓

📋 Rule 49 — Quick Recap

  • Common Possession (jointly owned) → apostrophe only at LAST noun: "Ram and Shyam's car" ✓
  • Common Possession can take singular OR plural noun: "Shyam's car" ✓ | "Shyam's cars" ✓
  • Split Possession (separate ownership) → apostrophe at EACH noun: "Ram's and Shyam's cars" ✓
  • Split Possession MUST take plural noun and plural verb: "Ram's and Shyam's cars are…" ✓ | "Ram's and Shyam's car is…" ✗

it's vs its — Common Error of Possession

it's = it is / it has (contraction) · its = possessive adjective (उसका / उसकी)

→ The Core Difference
★ From the book (Rule 50): Some common errors of Possession.
We often make mistakes by using "it's" in place of "its". Both these terms have separate meanings.
"It's" denotes "It is" (or "it has"), while "its" denotes possession.
it's
= "it is" OR "it has"
(Contraction / shortened form)

it's cold = it is cold
it's been raining = it has been raining

NO possession meaning here
its
= Possessive Adjective
(Shows belonging / उसका / उसकी)

its book = उसकी किताब
its tail = उसकी पूंछ

Possession — no apostrophe needed
it's = it is / it has  ·  its = उसका/उसकी (possessive)
Test: Replace with "it is" — if it makes sense → use "it's". If not → use "its".
★ All Examples from Book
Example 1 — carrying ITS book (possession)
✗
The child is carrying it's book. [Incorrect]
"it's" = "it is" → "The child is carrying it is book" — doesn't make sense. ✗
✓
The child is carrying its book. [Correct]
"its" = possessive (उसकी किताब). Child is carrying its own book. ✓
Example 2 — IT'S called a miracle (it is)
✗
Its called a miracle. [Incorrect]
"its" = possessive → "its called" makes no sense as possession. ✗
✓
It's called a miracle. [Correct]
"It's" = "It is" → "It is called a miracle." ✓
Class Examples
✓
The dog wagged its tail. [Correct]
"its" = possessive (उसकी पूंछ) — dog's own tail. ✓
✓
It's raining heavily today. [Correct]
"It's" = "It is" → "It is raining heavily today." ✓
💡
The Quick Test:
Replace the word with "it is" in the sentence.
→ If the sentence still makes sense → use it's (contraction).
→ If the sentence becomes nonsense → use its (possessive).

"The child is carrying [it is] book" → doesn't make sense → use its ✓
"[It is] called a miracle" → makes sense → use it's ✓

📋 Rule 50 — Quick Recap

  • it's = contraction of "it is" / "it has" — NOT possessive
  • its = possessive adjective (उसका/उसकी) — NO apostrophe needed
  • "The child is carrying its book." ✓ (possession) | "carrying it's book" ✗
  • "It's called a miracle." ✓ (= it is) | "Its called a miracle." ✗
  • Quick test: Can you replace with "it is" and still make sense? → it's. Not? → its.

Rule 49 — Common vs Split Possession Possession

Key rule:Split Possession NEVER takes singular noun — at least 2 separate things exist.
