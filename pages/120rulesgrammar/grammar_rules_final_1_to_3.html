Each & Every

Difference between "Each" and "Every" — 4 key concepts

★ Primary Rule — Always followed by a Singular Verb

Both Each and Every are always followed by a singular verb. This is the most fundamental rule — no exceptions.

✓
Each student is very talented.
Singular verb "is" used. [Singular verb is used]
✓
Each of my students is very talented.
Singular verb "is" used even though "students" is plural. [Singular verb is used]
Why? In sentence (I), "each" performs the function of an Adjective (words used to describe or modify nouns/pronouns). The word it describes is the noun "student".

In sentence (II), "each" performs the function of a Pronoun (word used instead of a noun or noun phrase). In both cases, we use the singular verb "is".
→ Core Meaning & Approach

Both Each and Every refer to a plural number of people or items — but they differ in how they consider the group:

WordApproachHow to think of it
Each Individualistic Approach Refers to a plural number of people or items, but considers each person or item as an individual or dissimilar unit — it does not consider the group collectively.
Every General Approach Also refers to a multiple number of people or items, but considers these people or items as part of a similar collective group.
1 Concept 1 — Number of People / Things

Each can be used for two or more people / things.
Every can only be used for more than two people / things.

⚠️ Every CANNOT be used when there are only two.

Examples
✗
I have a book in every hand.
WRONG — There are only 2 hands. "Every" cannot be used for just two things.
✓
I have a book in each hand.
CORRECT — "Each" works for two (both hands), so it is used here.
💡
Memory trick: Think "Every = Everyone in a crowd (3+)". If there are only 2, always use each.
More Examples from the Book
✓
Each person on this earth has to die one day.
More than two persons → Both "each" and "every" can be used here.
✓
Every person I know has watched the movie.
More than two persons → "every" is correct. [More than two persons]
✓
He has a book in each hand.
Only 2 hands → "each" is correct. [Correct]
✗
He has a book in every hand.
Only 2 hands → "every" cannot be used with exactly two things. [Incorrect] — Every cannot be used with two things.
2 Concept 2 — Pronoun vs. Adjective

First, understand the difference:

TermWhat it does
PronounUsed in place of a noun (replaces the noun)
AdjectiveTells us about a noun (always followed by a noun)

Each can be used as both a Pronoun AND an Adjective.
Every can be used only as an Adjective — never as a Pronoun.

Examples
✓
Each of my students is working very hard.
Here "Each" is a Pronoun — it stands alone (no noun immediately after it). Followed by "of".
✓
Each student is working very hard.
Here "Each" is an Adjective — it directly describes the noun "student".
✗
Every of my students is very talented.
WRONG — "Every" is NOT a pronoun. It cannot stand alone followed by "of". You must put a noun directly after it.
✓
Every student is very talented.
CORRECT — "Every" is directly followed by the noun "student" (used as adjective only).
Rule to remember: If you see "Every of…" in any sentence → it is always wrong. Replace with "Each of…" OR rewrite as "Every + noun".
3 Concept 3 — Abstract Nouns

What is an Abstract Noun?
A noun that you cannot touch, cannot see — you can only feel it. Examples: love, poverty, laughter, reason, happiness, pain, hope.

Each — CANNOT be used with Abstract Nouns.
Every — CAN be used with Abstract Nouns.

Examples
✓
They have every reason to be happy with their new home.
CORRECT — "reason" is an abstract noun. "Every" can be used with abstract nouns. [Correct]
✗
They have each reason to be happy with their new home.
WRONG — "reason" is an abstract noun (you can't touch or see a reason). "Each" cannot be used with abstract nouns. [Incorrect]
4 Concept 4 — Verb, Adjective & Noun Form Rules

This applies to all four words: Each / Every / Either / Neither

Either = one of the two  |  Neither = none of the two

🔑 Golden Rule:

What followsForm
Verb (after Each / Every / Either / Neither)Always Singular (is / was / has / does…)
Possessive Adjective (his / her / its / my…)Always Singular (his / her / its — NOT their)
📌 See Rule 2 for a full explanation of what Possessive Adjectives are
Examples — Verb must be Singular
✗
Each student is preparing for their exam.
WRONG — "their" is plural. Since "each" is used, the possessive must be singular: his / her.
✓
Each student is preparing for his/her exam.
CORRECT — Singular possessive adjective used.
Noun Form — Directly after Each / Every vs. after "of the"
StructureNoun formExample
Each / Every + directly + Noun Singular Each student | Every child
Each / Every + of the + Noun Plural Each of the students | Every one of the children
💡
Quick rule: Direct → Singular. After "of the" → Plural. Simple as that!

📋 Rule 1 — Quick Recap

  • Each = 2 or more; Every = more than 2 only (not for 2)
  • Each = Pronoun + Adjective; Every = Adjective only (never "Every of…")
  • Each cannot be used with Abstract Nouns; Every can be
  • After Each / Every / Either / Neither → Verb & Possessive Adjective are always SINGULAR
  • Each/Every + direct noun → Singular noun; Each/Every + of the → Plural noun

✗

✓

2

Both

3 key concepts — connectors, negation & possessive adjective placement

1 Concept 1 — Both is always used with "AND" (never "as well as")

When Both connects two people or things, it must always be followed by AND.
⚠️ Never use "as well as" with Both.

✗
Both Ram as well as Shyam are coming.
WRONG — "as well as" cannot be used with "both".
✓
Both Ram and Shyam are coming.
CORRECT — Always use "and" after both.
💡
Formula: Both + [A] + and + [B] — This is the only correct pattern.
2 Concept 2 — "Not" CANNOT be used with Both

Both and Not can NEVER appear together in a sentence.
If you want to say "neither of them", use Neither…Nor instead.

✗
Both Ram and Shyam are not coming.
WRONG — "not" cannot be used with "both".
✓
Neither Ram nor Shyam is coming.
CORRECT — Replace "Both…not" with "Neither…nor".
✗
This is a war which both sides cannot win.
WRONG — "both" and "not/cannot" cannot appear together. [Incorrect]
✓
This is a war which neither side can win.
CORRECT — "neither" replaces "both…not". Note: "neither" goes before a singular countable noun ("side", not "sides"). [Correct]
Important: With Neither…Nor, the verb agrees with the subject that is closest to it. Since "Shyam" is singular → is is used (not "are").

Extra Note: "Neither" allows us to make a negative statement about two people or things at the same time. "Neither" goes before singular countable nouns. We use it to say "not either" in relation to two things.
🔍 What is a Possessive Adjective? Key Concept

A Possessive Adjective is a word that shows ownership or belonging — it tells you whose something is. It always sits before a noun to describe it.

Simple way to remember: If a phone is mine → I say "my phone". The word my is the Possessive Adjective because it shows the phone belongs to me.

mywith → I
"my bag"
yourwith → You
"your book"
hiswith → He
"his pen"
herwith → She
"her dress"
itswith → It
"its tail"
ourwith → We
"our house"
theirwith → They
"their school"
Important: A Possessive Adjective always comes before a noun — "my book", "her dress", "their school". If you remove the noun and it still makes sense alone (e.g. "The book is mine"), that is a Possessive Pronoun, not a Possessive Adjective. In these grammar rules, whenever "possessive adjective" is mentioned, it refers to: my / your / his / her / its / our / their.
📌 Where does this concept appear in these 3 Rules?
Rule 1 — Concept 4
After Each / Every / Either / Neither, the possessive adjective must always be singular. Use his or her, never their.
Example: "Each student must submit his/her assignment." ✓ — not "their"
Rule 2 — Concept 3
When used with Both, the possessive adjective must always come after "both", never before it.
Example: "Both my friends came." ✓ — not "My both friends"
3 Concept 3 — Possessive Adjective comes AFTER Both

A Possessive Adjective (my, your, his, her, its, our, their — see the full explanation above ↑) shows ownership. When used with Both, it must always come AFTER "both" — never before it.

✗
My both students are brilliant.
WRONG — Possessive adjective "my" cannot come before "both".
✓
Both my students are brilliant.
CORRECT — "my" comes after "both".
💡
Formula: Both + Possessive Adjective + Noun  →  "Both my books", not "My both books".

📋 Rule 2 — Quick Recap

  • Both + and (never "as well as") → Both Ram AND Shyam
  • Both + NOT = WRONG → Replace with Neither…Nor
  • Possessive adjective always comes after Both → "Both my friends" ✓
  • With Neither…Nor → verb agrees with the nearest subject

✏️ Practice — Correct or Wrong?

✗
Both Ram and Shyam are not going.
WRONG — "not" with "both" is incorrect. Use "Neither Ram nor Shyam is going."
✓
Neither Ram nor Shyam is going.
CORRECT ✓
✗
My both sisters are doctors.
WRONG — Possessive before "both". Should be: "Both my sisters are doctors."
✓
Both my sisters are doctors.
CORRECT ✓

Relative Pronouns — Who, Whom, Which & That

Usage of Who, Whom, Which, That + Whoever vs. Whomever

→ Foundation — Subjective & Objective Case

To understand Who vs Whom, you must first understand these two cases:

CasePosition in sentenceSimple rule
Subjective Comes before the verb The subject of the verb — the one doing the action
Objective Comes after the verb The object — the one receiving the action
Pronoun Lists
Subjective Pronouns
  • I
  • We
  • He
  • She
  • It
  • They
  • You
Objective Pronouns
  • Me
  • Us
  • Him
  • Her
  • It
  • Them
  • You

Example: He met her. → "He" = Subjective (before verb "met"), "her" = Objective (after verb "met").

1 Who vs. Whom — The Core Rule
WordCaseEquivalent pronounPosition
Who Subjective He / She / They (doer) Before the verb in its clause
Whom Objective Him / Her / Them (receiver) After the verb in its clause
💡
The Substitution Trick: Replace the blank with he/she or him/her.
If he/she fits → use Who.
If him/her fits → use Whom.
Worked Examples
Example 1: She met the person ___ says that he is God.

Fill the blank: "He says…" fits (he = subjective) → Who
✓ She met the person who says that he is God.
Example 2: She met the person ___ God likes.

Fill the blank: "God likes him" fits (him = objective) → Whom
✓ She met the person whom God likes.
Example 3: She met him ___ I think was arrested yesterday.

Fill: "He was arrested" fits (he = subjective) → Who
✓ She met him who I think was arrested yesterday.
Example 4: She met him ___ the police arrested.

Fill: "The police arrested him" fits (him = objective) → Whom
✓ She met him whom the police arrested.
2 Who & Whom — Used for Whom?

Who and Whom are used for:

  • ✅ Human beings (men, women, children)
  • ✅ Domestic / pet animals (dogs, cats, cows — animals kept and named by people)
  • ❌ Non-domestic animals, objects, places → use Which / That instead
Examples
✓
The witnesses whom I interviewed gave conflicting evidence.
Used for people. [Used for people]
✓
The people who just boarded the plane are in a rock band.
Used for people. [Used for people]
✓
The vacuum scared our dog Rocky, who was sleeping on the rug.
Rocky is a named/domestic pet → Who/Whom is correct for domestic animals. [Used for a pet]
3 "Which" — Used for Non-Domestic Animals & Non-Living Things

"Which" is used for animals in general or things — i.e., Non-domestic Animals and Non-living things.

Examples
✓
Vinita visited the park with a dog, which likes to chase squirrels.
A non-domestic / general animal → "which" is used. [Used for an animal]
✓
The bomb, which killed 15 people, was the worst of a series of terrorist outrages.
A non-living thing → "which" is used. [Used for a non-living thing]
4 "That" — No Restrictions (People, Animals or Things)

"That" can be used for people, animals or things — there are no restrictions on what it can refer to.

Examples
✓
The car that Jasprit bought runs on electricity and gas.
[Used for a thing]
✓
The musician that won the award is Indian.
[Used for a person]
✓
The dog that is chasing the squirrels belongs to Brijesh.
[Used for an animal]
💡
Why was "That" created?
Consider this sentence: "I saw the man and the box ___ you were talking of."

— You cannot use "who" here, because "box" is a non-living thing.
— You cannot use "which" here, because "man" is a person.

So what do you use? → "That" — because "that" has no restrictions. It can refer to both a person and a thing at the same time. This is exactly why "that" was invented — for situations where neither "who" nor "which" alone works.

✓ "I saw the man and the box that you were talking of."
★ Special NOTE: The words all, any, the only, the same, none, much, no, the little, the few, something, anything, nothing and superlative degrees are all followed by "that" only. They cannot be followed by "who" or "which".

✗
This is the same book which you were reading.
WRONG — "same" must be followed by "that", not "which". [Incorrect]
✓
This is the same book that you were reading.
CORRECT — "same" is followed by "that". [Correct]
✓
This is the same person that I want to marry.
CORRECT — "same" → must use "that". [Correct]
✗
He spent up the little amount which he had.
WRONG — "the little" must be followed by "that", not "which". [Incorrect]
✓
He spent up the little amount that he had.
CORRECT — "the little" → must use "that". [Correct]
✓
This is the best that I can do for you.
CORRECT — superlative "best" is followed by "that". [Correct]
5 Antecedent — The Key to Choosing Who/Whom vs. Whoever/Whomever

Before understanding Whoever/Whomever, you must understand what an Antecedent is.

What is an Antecedent?
An Antecedent is the noun or pronoun that comes before "who" or "whom" in a sentence — the word that "who/whom" is referring back to.

Example: "She saw him who is a rockstar."
Here, "him" is the antecedent — it is the noun/pronoun present right before "who".
The Golden Rule
SituationWhat to useExample
A noun or pronoun (antecedent) IS present before the blank Who or Whom "She saw him ___ is a rockstar." → who (him = antecedent present)
No noun or pronoun (no antecedent) before the blank Whoever or Whomever "She saw ___ is a rockstar." → whoever (no antecedent)
💡
Simple trick:
👉 Antecedent present (noun/pronoun before the blank) → Who / Whom
👉 No antecedent (no noun/pronoun before the blank) → Whoever / Whomever

Think of "ever" as the word you add when there is no antecedent. No antecedent → add "ever" → Whoever / Whomever.
Examples — With Antecedent (use Who / Whom)
✓
She saw him who is a rockstar.
"him" = antecedent is present → use Who (subjective, as "he is a rockstar"). ✓
✓
They met them who work very hard.
"them" = antecedent is present → use Who (subjective, as "they work very hard"). ✓
Examples — Without Antecedent (use Whoever / Whomever)
✓
She saw whoever is a rockstar.
No antecedent before the blank → add "ever" → use Whoever (subjective). ✓
✓
Please invite whomever you choose.
No antecedent before the blank + objective (you choose him) → use Whomever. ✓
✓
I am eager to work with whomever Dale selects as my partner.
No antecedent before the blank + objective (Dale selects her) → use Whomever. ✓
6 "Whoever" vs. "Whomever" — Detailed Examples
WordTypeFunction
Whoever Subjective pronoun Serves as the subject of a verb
Whomever Objective pronoun Serves as the object of a verb or a preposition
Extra Rule for Whoever: "Whoever" is also correct as a subject complement, used with linking verbs such as is, are, and will be.
Whoever — Examples (Subjective)
✓
Whoever is hungry can have the leftover pizza.
"Whoever" is the subject of the verb is — like "He is hungry." → Subjective → Whoever.
✓
Give the package to whoever comes for it.
"Whoever" is the subject of the verb comes — like "He comes for it." → Subjective → Whoever.
✓
I am happy with whoever wins.
"Whoever" is the subject of the verb wins — like "He wins." → Subjective → Whoever.
Whoever as Subject Complement (with linking verbs)
✓
Whoever it was did not leave her name.
Subject complement with linking verb "was". → Whoever is correct.
✓
Whoever they are, I like their confidence.
Subject complement with linking verb "are". → Whoever is correct.
Whomever — Examples (Objective)
✓
Please invite whomever you choose.
"Whomever" is the object of the verb choose — like "You choose him." → Objective → Whomever.
✓
Whomever Kate marries is none of our business.
"Whomever" is the object of the verb marries — like "Kate marries him." → Objective → Whomever.
✓
I am eager to work with whomever Dale selects as my partner.
"Whomever" is the object of the verb selects — like "Dale selects her." → Objective → Whomever.
💡
Quick trick: Same logic as Who vs Whom.
Substitute he/she → fits? → use Whoever.
Substitute him/her → fits? → use Whomever.

📋 Rule 3 — Quick Recap

  • Who = Subjective case → where He/She/They fits → used for humans & domestic/named animals
  • Whom = Objective case → where Him/Her/Them fits → used for humans & domestic/named animals
  • Which = used for non-domestic animals and non-living things
  • That = no restrictions — people, animals, or things. Also mandatory after: all, any, the only, the same, none, much, no, the little, the few, something, anything, nothing, and superlative degrees. Also use "that" when referring to a person AND a thing together (e.g. "the man and the box that…")
  • Antecedent rule: If a noun/pronoun (antecedent) is present before the blank → use Who / Whom. If NO antecedent → use Whoever / Whomever
  • Whoever = Subjective pronoun (subject of a verb, or subject complement with linking verbs) — used when NO antecedent
  • Whomever = Objective pronoun (object of a verb or preposition) — used when NO antecedent
  • Quick check: He fits → Who/Whoever. Him fits → Whom/Whomever. No antecedent? → add "ever".

Rule 1 — Each & Every 4 Concepts
Primary Rule:Both Each and Every are always followed by a singular verb. No exceptions.
Concept 1 — Numbers:Each = 2 or more people/things.  Every = more than 2 only (NOT for exactly 2). E.g. "Each hand" ✓ (2 hands), "Every hand" ✗ (only 2).
Concept 2 — Word function:Each = Pronoun + Adjective.  Every = Adjective only. So "Every of the students" ✗ — never use "Every of…"; say "Each of the students" ✓.
Concept 3 — Abstract nouns:Each CANNOT be used with Abstract Nouns (e.g. joy, reason, courage). Every CAN be used with Abstract Nouns. E.g. "Every reason to be happy" ✓.
Concept 4 — Verb & Possessive:After Each / Every / Either / Neither → verb is always singular AND possessive adjective is always singular (his/her — never "their"). E.g. "Each student must submit his/her assignment." ✓
Noun form rule:Each/Every + direct noun → Singular noun.  Each/Every + of the → Plural noun. E.g. "Each student" vs "Each of the students".
Quick Check: Each = individual focus. Every = group/general. Each works for 2+. Every needs 3+.
2
Rule 2 — Both 3 Concepts
Concept 1 — Connector:Both must always be paired with AND. NEVER use "as well as" with both. Formula: Both + [A] + and + [B]. E.g. "Both Ram and Shyam are coming." ✓
Concept 2 — Negation:"Not" CANNOT be used with Both. Replace "Both…not" with Neither…Nor. E.g. "Both Ram and Shyam are not going." ✗ → "Neither Ram nor Shyam is going." ✓
Neither…Nor — verb:With Neither…Nor, the verb agrees with the nearest subject. "Neither Ram nor Shyam is" (Shyam = singular, nearest).
Concept 3 — Possessive:Possessive adjective (my/your/his/her…) must come AFTER both, never before. Formula: Both + Possessive Adj + Noun. E.g. "Both my friends" ✓, "My both friends" ✗.
Possessive adjectives:my · your · his · her · its · our · their — words that show ownership, placed before a noun.
Summary: Both + and ✓  |  Both + not ✗ → use Neither…nor  |  Both + [PA] ✓  |  [PA] + Both ✗

Rule 3 — Who, Whom, Which & That 6 Concepts
Cases foundation:Subjective = before the verb (doer). Objective = after the verb (receiver). Subjective pronouns: I/We/He/She/They. Objective: Me/Us/Him/Her/Them.
Who vs Whom:Who = Subjective case (like He/She/They). Whom = Objective case (like Him/Her/Them).
Substitution trick: Replace blank with he/she → Who. Replace with him/her → Whom.
Who / Whom — used for:✅ Human beings.  ✅ Domestic / named pet animals.  ❌ Non-domestic animals / objects / places → use Which/That.
Which — used for:Non-domestic/general animals and non-living things. E.g. "a dog, which likes to chase squirrels" ✓.
That — no restriction:Can refer to people, animals, or things. Mandatory after: all, any, the only, the same, none, much, no, the little, the few, something, anything, nothing, superlative degrees. Also use when a person AND a thing appear together.
Antecedent rule:Antecedent present (noun/pronoun before the blank) → use Who / Whom.
No antecedent (no noun/pronoun before the blank) → use Whoever / Whomever. Think: no antecedent → add "ever".
Whoever vs Whomever:Whoever = Subjective (subject of verb, or subject complement with linking verbs is/are/will be). Whomever = Objective (object of verb or preposition). Same he/she vs him/her trick applies.
