68

whoever / whomever — Add "ever" when NO Noun or Pronoun Precedes who/whom

If a noun or pronoun comes before "who/whom" → use who/whom normally · If NO noun/pronoun precedes → add "ever" to make whoever/whomever

→ The Core Rule
★ Rule 68: When "who" or "whom" is NOT preceded by a noun or pronoun, we always add the word "ever" to it → making whoever or whomever. This is because "whoever/whomever" acts independently as the subject or object.
No noun/pronoun before "who/whom" → whoever / whomever
Noun/pronoun IS present before "who/whom" → keep who / whom
SituationUseExample
Noun/pronoun precedeswho / whom"The person who helped us…"
NO noun/pronoun precedeswhoever / whomever"Whoever helped us…"
★ Examples from Class
✓
Ram met whoever they have helped. [Correct]
No noun/pronoun before "who" → use "whoever". ✓
✓
He saw whoever was there. [Correct]
No noun/pronoun before "who" → "whoever". ✓
✗
Ram met who they have helped. [Incorrect]
No noun/pronoun before "who" → must add "ever" → "whoever". ✗
💡
Think of it this way:
"who" and "whom" are relative pronouns — they connect back to a noun or pronoun already mentioned.
If there's no noun or pronoun to refer to → they can't work alone → add "ever" to make them independent pronouns (whoever/whomever) that can stand on their own.

📋 Rule 68 — Quick Recap

  • Noun/pronoun before "who/whom" → keep who/whom
  • No noun/pronoun before "who/whom" → add "ever" → whoever/whomever
  • "Ram met whoever they have helped." ✓ · "He saw whoever was there." ✓
69

Question Tags — Always Opposite of the Statement

Positive statement → Negative tag · Negative statement → Positive tag · "है ना" = question tag in Hindi

→ What is a Question Tag?
★ Rule 69: A Question Tag is a short question added at the end of a statement to confirm that statement. In Hindi, we say "है ना?" — that "है ना" is the question tag.

"You are working hard, aren't you?" → "aren't you?" is the question tag.
"Tum mehnnat kar rahe ho, hai na?" → "hai na" is the question tag.
Question Tag = always OPPOSITE of the main statement
Positive statement → Negative tag · Negative statement → Positive tag
★ The Rule — Positive ↔ Negative
Case 1 — Positive Statement → Negative Question Tag

"I was doing it,"

✗ Negative Tag
"wasn't I?"

Positive statement → negative tag "wasn't I?" ✓
✓

Positive statement → negative tag "hasn't he?" ✓
✓

Positive statement → negative tag "aren't you?" ✓
✓

Positive statement → negative tag "can't she?" ✓
Case 2 — Negative Statement → Positive Question Tag
✗ Negative Statement
"I was not doing it,"
→
✓ Positive Tag
"was I?"
✓
I was not doing it, was I?
Negative statement → positive tag "was I?" ✓
✓

Negative statement → positive tag "has he?" ✓
✓
She cannot swim, can she?
Negative statement → positive tag "can she?" ✓
✓
You don't like coffee, do you?
Negative statement → positive tag "do you?" ✓
💡
How to form a Question Tag — 3 Steps:
1. Take the auxiliary verb from the main statement (is/are/was/were/has/have/had/do/does/did/can/will…)
2. If statement is positive → make it negative in the tag (add "not" = "n't")
3. If statement is negative → use the positive auxiliary
4. Use the pronoun that matches the subject (I → I, he → he, they → they)

📋 Rule 69 — Quick Recap

  • Question tag = short question at end of statement to confirm it (= "है ना?" in Hindi)
  • Positive statement → Negative tag: "I was doing it, wasn't I?" ✓
  • Negative statement → Positive tag: "I was NOT doing it, was I?" ✓
  • Tag uses the auxiliary verb of the statement + matching pronoun
70

"amn't I" Does NOT Exist — Always Use "aren't I" in Question Tags

There is no word like "amn't" in English · The correct question tag for "I am…" is always "aren't I?"

→ The Core Rule
★ Rule 70: There is no such word or phrase as "amn't I" in English. Whenever you need to write "amn't I" in a question tag (e.g., for "I am…"), you must always write "aren't I?" instead. This is the correct and accepted form.
"I am…, aren't I?" ✓  ·  "I am…, amn't I?" ✗
"amn't" is NOT a word — it does not exist in English
★ Examples
✗
I am working hard, amn't I? [Incorrect]
"amn't" does not exist. ✗
✓
I am working hard, aren't I? [Correct]
"aren't I?" is the accepted question tag for "I am…". ✓
✗
I am the best, amn't I? [Incorrect]
"amn't" does not exist. ✗
✓
I am the best, aren't I? [Correct]
✗
I am late, amn't I? [Incorrect]
✓
I am late, aren't I? [Correct]
⭐
Exception to the "opposite" rule for "I am":
Normally "I am + positive" → tag should use negative "am not I?" → But "am not I?" sounds very formal/archaic. So instead of "amn't I" (which doesn't exist), English uses "aren't I?" as the contracted form. This is the universally accepted standard form in modern English.
💡
Full Question Tag chart for "I am" statements:
✓ "I am working hard, aren't I?"
✓ "I am a good student, aren't I?"
✓ "I am not late, am I?" (negative statement → positive tag → "am I?" ✓)
✗ Never write "amn't I" — it is NOT a word.

📋 Rule 70 — Quick Recap

  • "amn't" does NOT exist in English — there is no such word
  • For "I am…" (positive) → question tag = "aren't I?": "I am working hard, aren't I?" ✓
  • For "I am not…" (negative) → question tag = "am I?": "I am not late, am I?" ✓
  • Exam: if you see "amn't I?" as an option — it is ALWAYS wrong. ✗
