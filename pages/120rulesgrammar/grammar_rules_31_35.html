Words Always Followed by Preposition "to"

All -ior ending words + prefer + known + married (in passive voice) → always take "to", never "by/over/with"

→ The Core Rule
★ From the book (Rule 31): There are certain words which are always followed by the preposition "to".

NOTE: Any word suffixed with "-ior" will always be followed by the preposition "to".
NOTE: The words "marry" and "known" are always followed by the preposition "to" in the Passive Voice.
→ All Words That Take "to"

senior

ulterior

anterior
पहले का / आगे का
posterior
पीछे का
exterior
बाहर का
interior
अंदर का
injurious
हानिकारक
addicted

ancillary

known

married
passive voice में
Memory trick: All words ending in -ior (superior, inferior, junior, senior, ulterior, exterior, interior, posterior, anterior…) automatically take "to". Plus: prefer, injurious, addicted, ancillary, known (passive), married (passive).
★ All Examples from Book & Class
prefer → always "to", never "over"
✗
I prefer tea over coffee. [Incorrect]
"prefer" never takes "over". ✗
✓
I prefer tea to coffee. [Correct]
"prefer" always takes "to". ✓
✓
Would you prefer to pay by cash, cheque, or credit card? [Correct]
prefer → to. ✓
known → passive voice always takes "to", NOT "by"
✗
I am known by him. [Incorrect]
"known" in passive takes "to", not "by". ✗
✓
I am known to him. [Correct]
"known" → "to". ✓
married → passive voice always takes "to"
✗
Sita is married with Ram. [Incorrect]
"married" (passive) takes "to", not "with". ✗
✓
Sita is married to Ram. [Correct]
"married" → "to". ✓
Active voice is fine: "Ram married Sita." ✓ — Passive voice requires "to": "Sita is married to Ram." ✓
-ior words → always "to"
✓
For babies, breastfeeding is superior to bottle-feeding. [Correct]
superior (-ior) → to. ✓
✓
They promoted a man who was junior to me. [Correct]
junior (-ior) → to. ✓
✓
Paragraph 19 was merely ancillary to paragraph 16. [Correct]
ancillary → to. ✓
injurious / addicted → always "to"
✓
Too much alcohol is injurious to your health. [Correct]
injurious → to. ✓
✓
He is addicted to smoking. [Correct]
addicted → to. ✓
elder / older — Common Trap!
⚠️
"elder" does NOT end in "-ior", so it does NOT take "to".
"He is elder than me." ✓ (elder takes "than", not "to"). Only words with the literal "-ior" suffix use "to".

📋 Rule 31 — Quick Recap

  • All words ending in -ior → always followed by "to": superior/inferior/junior/senior/ulterior/exterior/interior/posterior/anterior
  • Also: prefer, injurious, addicted, ancillary → take "to"
  • known and married in passive voice → take "to" (NEVER "by/with")
  • "I prefer tea to coffee" ✓ | "I am known to him" ✓ | "Sita is married to Ram" ✓
  • Trap: "elder" does NOT take "to" → uses "than" instead

Use of Phrases "as…as" and "so…as"

"as…as" = positive & negative both · "so…as" = ONLY negative sentences · Always use Positive Degree adjective

→ The Core Rule
★ From the book (Rule 32): The use of phrase "as…as" and "so…as".
These blanks are filled up with the adjectives of positive degree.

"as…as" can be used to denote both positive AND negative sentences.
"so…as" can only be used to denote NEGATIVE sentences only.
as … as — Positive & Negative both ✓
  • He is as good as Ram. ✓ (Positive)
  • He is not as good as Ram. ✓ (Negative)
so … as — ONLY Negative sentences ✓
  • He is not so good as Ram. ✓ (Negative only)
  • He is so good as Ram. ✗ (WRONG — positive)
as + [Positive Degree Adjective] + as → both positive & negative ✓
so + [Positive Degree Adjective] + as → ONLY in negative sentences ✓
★ All Examples from Book & Class
as…as — CORRECT in both positive and negative
✓
He is as good as Ram. [Correct]
Positive sentence + "as…as". ✓
✓
He is not as good as Ram. [Correct]
Negative sentence + "as…as". ✓
✓
If we pass on the information, he's as good as dead. [Correct]
"as…as" in positive sentence. ✓
so…as — CORRECT only in negative
✓
He is not so good as Ram. [Correct]
Negative sentence + "so…as". ✓
✓
I'm not so desperate as to agree to that. [Correct]
Negative sentence + "so…as". ✓
so…as in POSITIVE sentence — always wrong
✗
He is so good as Ram. [Incorrect]
"so…as" can NEVER be used in a positive sentence. ✗
✗
Ram is so good as Shyam. [Incorrect]
"so…as" with positive sentence = always wrong. ✗
💡
Quick Test: Is the sentence negative (has "not")?
→ Yes → both "as…as" and "so…as" are correct.
→ No (positive) → ONLY "as…as" is correct. "so…as" is NEVER correct in positive sentences.

📋 Rule 32 — Quick Recap

  • Both "as…as" and "so…as" use the Positive Degree adjective (good, not better)
  • "as…as" → positive sentences ✓ AND negative sentences ✓
  • "so…as" → ONLY negative sentences ✓ | NEVER positive sentences ✗
  • "He is as good as Ram" ✓ | "He is not so good as Ram" ✓ | "He is so good as Ram" ✗
33

"times" — Never Followed by Comparative Degree

With "times" → always Positive Degree · Without "times" → use Comparative Degree normally

→ The Core Rule
★ From the book (Rule 33): In a sentence depicting comparison, whenever the word "times" comes up, it is never followed by a comparative degree. We always use the Positive Degree with "times".

Why? Because "times" itself already shows comparison — so adding a comparative degree (-er / more) is redundant and wrong.
WITH "times"
→ use Positive Degree
three times as large as ✓
three times as heavy as ✓
WITHOUT "times"
→ use Comparative Degree
larger than ✓
heavier than ✓
★ All 6 Examples from Book
WITH "times" + Comparative Degree → WRONG
✗
(I) This book is three times larger than that book. [Incorrect]
"times" is present → cannot use comparative "larger". Should be "as large as". ✗
✗
(II) This book is three times heavier than that book. [Incorrect]
"times" is present → cannot use "heavier". Should be "as heavy as". ✗
WITHOUT "times" + Comparative Degree → CORRECT
✓
(III) It is twice cheaper than that. [Correct]
"twice" is used, NOT the word "times" — so comparative is fine. ✓
✓
(V) It is twice costlier than that. [Correct]
"twice" used without "times" → comparative degree is fine. ✓
WITH "times" + Positive Degree (as…as) → CORRECT
✓
(IV) This book is three times as large as that book. [Correct]
"times" + positive degree "as large as". ✓
✓
(VI) It is two times as costly as that. [Correct]
"times" + positive degree "as costly as". ✓
From the book: In sentence (I), the word "times" and the comparative degree "larger" cannot be used together. The sentence should be rewritten either as sentence (IV) to make it grammatically correct.
In sentences (III) and (V), there is no use of the word "times" and thus we can use the comparative degree to bring up the comparisons.
💡
The Rule in One Line:
TIMES used → as [positive] as (three times as large as) ✓
TIMES NOT used → [comparative] than (larger than) ✓

✏️ Correct or Incorrect?

✗
He is three times stronger than me.
"times" + comparative "stronger" = wrong. → "three times as strong as me" ✗
✓
He is three times as strong as me.
"times" + positive "as strong as" ✓
✓
This car is twice faster than that one.
No "times" word → comparative "faster" is fine. ✓

📋 Rule 33 — Quick Recap

  • "times" itself shows comparison → so never add comparative degree (-er / more) with it
  • With "times" → use as [positive] as: "three times as large as" ✓
  • Without "times" → use comparative than: "twice cheaper than" ✓ | "twice costlier than" ✓
  • "three times larger than" ✗ | "three times heavier than" ✗

Common Errors of Narration — Indirect Speech

Indirect Speech is ALWAYS a statement · Helping verb always comes AFTER subject in statements

→ Statement vs Question Structure
★ From the book (Rule 34): Some common errors of Narration.
Indirect Speech is always a statement. Hence a helping verb comes after a subject in the case of a statement and before a subject in the case of a question.
Structure of a STATEMENT
Subject + Helping Verb + Main Verb + …

Examples:
I am teaching English.
She is doing her work.

→ Helping verb comes AFTER subject.
Structure of a QUESTION
[WH word] + Helping Verb + Subject + Main Verb + …

Examples:
Am I teaching English?
Why are you gossiping?

→ Helping verb comes BEFORE subject.
⚠️
Indirect Speech is ALWAYS a statement, NEVER a question.
This means → in all indirect speech, the helping verb must come after the subject. The question structure (helping verb before subject) is NEVER used in indirect speech.
★ All Examples from Book & Class
Interrogative reported speech (most common error)
✗
He asked me where are you going. [Incorrect]
Indirect speech → statement structure needed. Helping verb "are" cannot come before subject "you". ✗
✓
He asked me where I was going. [Correct]
Subject "I" before helping verb "was". Statement structure. ✓
✗
He asked me what was my name. [Incorrect]
Helping verb "was" before subject "my name". Question structure → wrong in indirect speech. ✗
✓
He asked me what my name was. [Correct]
Subject "my name" before helping verb "was". ✓
No Error — already in statement order (Tricky!)
✓
He asked me where I was going. [Correct — No Error]
Subject "I" comes before "was". Statement order → correct. ✓ (Many students wrongly change this.)
Other types of indirect speech — all are statements
✓
She says that she is a little bit nervous. [Correct]
Assertive sentence in indirect speech → statement. Subject "she" before "is". ✓
✓
The teacher ordered me to shut the door. [Correct]
Imperative sentence → no subject in direct speech → "to + V1" in indirect. ✓
✓
He exclaimed with great joy that he had won the match. [Correct]
Exclamatory → statement. Subject "he" before "had". ✓
✓
We asked him if he was ill. [Correct]
Interrogative → "if/whether" + statement. Subject "he" before "was". ✓
✓
He prayed that I might live long. [Correct]
Optative → statement. Subject "I" before "might". ✓
✓
She wished she had been rich. [Correct]
Optative → statement. Subject "she" before "had been". ✓

📋 Rule 34 — Quick Recap

  • Indirect Speech is always a statement — NEVER a question
  • Statement: Subject + Helping Verb (subject comes first)
  • Question: Helping Verb + Subject (question structure — NEVER used in indirect speech)
  • "He asked me where I was going" ✓ | "He asked me where was I going" ✗
  • "He asked me what my name was" ✓ | "He asked me what was my name" ✗
  • All types (assertive, interrogative, imperative, exclamatory, optative) → indirect speech is always a statement

Verb Agrees with the Nearest Subject

either/or · neither/nor · not only/but also · or · nor — verb always matches the nearest (closest) subject

→ The Core Rule
★ From the book (Rule 35): If two subjects are connected by the words "or", "nor", "neither/nor", "either/or", or "not only/but also", we use the verb according to the nearest subject.
either … or neither … nor not only … but also or nor
[Subject 1] {connector} [Subject 2 — NEAREST] + Verb matching Subject 2
💡
Quick Rule: Ignore Subject 1. Look at Subject 2 (the one closest to the verb).
→ Subject 2 singular → Singular verb (is / has / does)
→ Subject 2 plural → Plural verb (are / have / do)
★ All Examples from Book & Class
either … or
✓
Either the bears or the lion has escaped from the zoo. [Correct]
Nearest subject = "lion" (singular) → singular verb "has". ✓
✓
Either the lion or the bears have escaped from the zoo. [Correct]
Nearest subject = "bears" (plural) → plural verb "have". ✓
neither … nor
✓
Neither the lion nor the bears have escaped from the zoo. [Correct]
Nearest subject = "bears" (plural) → plural verb "have". ✓
✓
Neither the bears nor the lion has escaped from the zoo. [Correct]
Nearest subject = "lion" (singular) → singular verb "has". ✓
not only … but also
✓
Not only the teachers but also the Principal is happy. [Correct]
Nearest subject = "Principal" (singular) → singular verb "is". ✓
✓
Not only dogs but also cats are available at the animal shelter. [Correct]
Nearest subject = "cats" (plural) → plural verb "are". ✓
💡
Pattern to always remember:
Neither the lion nor the bears have … → bears (plural) closest → "have" ✓
Either the bears or the lion has … → lion (singular) closest → "has" ✓

✏️ Practice — Choose Correct Verb

✓
Either Ram or his brothers are responsible.
Nearest = "brothers" (plural) → "are" ✓
✓
Neither Ram nor Shyam is available.
Nearest = "Shyam" (singular) → "is" ✓
✓
Not only the students but also the teacher was surprised.
Nearest = "teacher" (singular) → "was" ✓
✗
Not only the students but also the teacher were surprised.
Nearest = "teacher" (singular) → should be "was", not "were" ✗

📋 Rule 35 — Quick Recap

  • Connectors: either/or · neither/nor · not only/but also · or · nor
  • Verb ALWAYS agrees with the nearest subject (the second / closer one)
  • "Either bears or the lion has…" ✓ (lion = singular = nearest) | "Either lion or the bears have…" ✓ (bears = plural = nearest)
  • "Not only teachers but also the Principal is…" ✓ | "Not only dogs but also cats are…" ✓
