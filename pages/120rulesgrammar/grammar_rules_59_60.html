59

"start / begin / commence" → always use "on" · NEVER "from"

"from" = continuation (ongoing) · start/begin/commence = one-time event → one specific day → "on"

→ The Core Rule
★ From the book (Rule 59): We do not use the preposition "from" with the word "start".

In the first sentence, the preposition "from" gives the meaning that my exams will start on Monday, it will again start on Tuesday and it will continue doing so. Thus, an event started once cannot start again and again. To avoid this error, we use the correct preposition "on" which implies that the event has started on Monday only.

*NOTE: "from" implies continuation.
❌ start/begin/commence + from
WRONG — "from" implies ongoing/continuous, but starting is a one-time event.

✗ "My exams start from Monday."
✗ "He will start his preparation from Monday."
✗ "The exam commences from Monday."
✓ start/begin/commence + on
CORRECT — "on" refers to one specific day. Starting happens once.

✓ "My exams start on Monday."
✓ "He will start his preparation on Monday."
✓ "The exam commences on Monday."
start / begin / commence / launch → always "on" + [day]  ·  NEVER "from" + [day]
from = continuation (something ongoing over time) · on = one specific day
💡
Why NOT "from"?
"from Monday" = something that continues from Monday onward (e.g., preparing from Monday for weeks).
But something can only start once — it cannot restart every day. So "start from Monday" implies it starts again and again, which is wrong.
"Start ON Monday" = it begins on that one specific day. ✓

📋 Rule 59 — Quick Recap

  • start / begin / commence → always use "on" + day: "exams start on Monday" ✓ | "exams start from Monday" ✗
  • "from" = continuation (something ongoing): "He will prepare from Monday." ✓ (no start word)
  • Starting is a one-time event → one day → "on" · Never "from" with start/begin/commence
60

"to + V1" (Infinitive) vs "to + V1+ing" (Gerund) — 12 Special Words

Normally: to + V1 · Special cases: to + V1+ing (gerund) · 12 words that ALWAYS take to + V1+ing

→ The Core Rule — Infinitive vs Gerund
★ From the book (Rule 60): In a general case of infinitive we always use the form "To + V1". But in certain special cases, we use the form "To + V1+ing".

*NOTE: When we use the form "V1+ing" as a noun, it is called Gerund.
to + V1 (Infinitive)
General case — used when showing an action or purpose.

✓ "I want to go."
✓ "She decided to leave."
✓ "He came to meet us."

(Action-focused — answers "to do what?")
to + V1+ing (Gerund)
Special case — used when "to" is followed by a noun (V1+ing used as noun).

✓ "I am used to teaching."
✓ "She is accustomed to being treated well."
✓ "I look forward to meeting you."

(Noun-focused — answers "used to what?")
Normal: to + V1  ·  12 Special Words: always to + V1+ing (Gerund)
Test: Does "to" answer "what?" → if yes, use V1+ing (gerund)
★ The 12 Special Words — All Take "to + V1+ing"
★ From the book: There are certain words which always follow "To + V1+ing" form:
(1)
be used to
"I am used to teaching."
(2)
accustomed to
"Not accustomed to being treated like this."
(3)
addicted to
"Addicted to teaching."
(4)
with a view to
"With a view to increasing profits."
(5)
look forward to
"Look forward to hearing from you."
(6)
given to
"Given to staying in bed till lunch."
(7)
prone to
"Prone to making tactless remarks."
(8)
devoted to
"Devoted herself to getting justice."
(9)
in addition to
"In addition to working hard…"
(10)
confess to
"He confessed to stealing the money."
(11)
disposed to
"Disposed to accepting the offer."
(12)
come to
"She came to loving the city."

📋 Rule 60 — Quick Recap

  • General rule: to + V1 (infinitive): "She decided to leave." ✓
  • 12 special words ALWAYS take to + V1+ing (gerund):
  • be used to · accustomed to · addicted to · with a view to · look forward to · given to
  • prone to · devoted to · in addition to · confess to · disposed to · come to
  • "I am used to teaching" ✓ | "I am used to teach" ✗ · "Look forward to hearing" ✓ | "to hear" ✗
  • Gerund = V1+ing used as a noun
