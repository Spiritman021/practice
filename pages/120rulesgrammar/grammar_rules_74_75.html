74

Collective Nouns in Question Tags — Same Rule as in Statements

All units doing the SAME thing → singular → "isn't it?" · All units doing DIFFERENT things → plural → "aren't they?" — identical to subject-verb agreement rule

→ The Core Rule (Recall from SVA)
★ Rule 74: The collective noun concept in question tags is exactly the same as in subject-verb agreement (covered earlier):

• If all units are doing the same thing → treated as singular → tag: "isn't it?"
• If all units are doing different things → treated as plural → tag: "aren't they?"
✓ Same action → SINGULAR → "isn't it?"
All units working together as one unit → treat as singular → tag uses "it"

"The army is marching towards the Himalayas,"
isn't it? ✓

"The jury has taken the decision,"
hasn't it? ✓
Different actions → PLURAL → "aren't they?"
All units doing separate/different things → treat as plural → tag uses "they"

"The army are changing their dresses,"
aren't they? ✓

"The audience are taking their seats,"
aren't they? ✓
★ All Examples from Class
Singular (same action) — tag uses "it"
✓
The army is marching towards the Himalayas, isn't it? [Correct]
Whole army doing ONE same thing (marching) → singular → "isn't it?" ✓
✓
The jury has taken the decision, hasn't it? [Correct]
Jury took ONE collective decision → singular → "hasn't it?" ✓
✓
The committee has passed the resolution, hasn't it? [Correct]
One resolution passed together → singular → "hasn't it?" ✓
Plural (different actions) — tag uses "they"
✓
The army are changing their dresses, aren't they? [Correct]
Each soldier changing their OWN dress (different clothes) → plural → "aren't they?" ✓
✓
The audience are taking their seats, aren't they? [Correct]
Each person sitting on a DIFFERENT seat → plural → "aren't they?" ✓
✓
The army changed their uniforms, didn't they? [Correct]
Different uniforms for different members → plural → "didn't they?" ✓
💡
Quick Test:
Ask: "Is everyone doing the SAME one thing, or DIFFERENT things?"
→ SAME thing (marching together, one decision) → singular → "it" → "isn't it? / hasn't it?"
→ DIFFERENT things (own dresses, own seats) → plural → "they" → "aren't they? / haven't they?"

📋 Rule 74 — Quick Recap

  • Same rule as SVA — no difference in question tags
  • Same action → singular → tag = "it": "The army is marching, isn't it?" ✓ · "The jury decided, didn't it?" ✓
  • Different actions → plural → tag = "they": "The army are changing dresses, aren't they?" ✓ · "The audience are taking seats, aren't they?" ✓
75

Imperative Sentences — 4 Types · Proposal (let's/let us) → Tag = "shall we?"

Request · Order · Advice · Proposal — Whenever "let's" or "let us" is used (= proposal) → the question tag is ALWAYS "shall we?"

→ 4 Types of Imperative Sentences
★ Rule 75: Imperative sentences give a command, request, advice, or proposal. There are exactly 4 types. Each type has a different question tag.
(1) Request
Please + verb
"Please give me a book."
"Please help me with this."
(2) Order
Direct command
"Close the door." / "Shut the door."
"Sit down." / "Be quiet."
(3) Advice
Suggestion / guidance
"Respect your parents."
"Obey your elders."
⭐ (4) Proposal — KEY for this rule!
Let's / Let us + verb
"Let's go for a walk."
"Let us consider all possibilities."
"Let's go out to dinner."
★ Proposal → Question Tag = "shall we?" — ALWAYS
★ From the book (Rule 75): Whenever "let's" or "let us" is used (= proposal/suggestion to do something together), the question tag is ALWAYS "shall we?" — without exception.
Let's / Let us + … ,
shall we?
"Let's…" OR "Let us…" → Question tag = "shall we?" ALWAYS
NEVER "don't we?" · NEVER "won't we?" · NEVER "will we?"
Examples — All from class
✓
Let's go for a walk, shall we? [Correct]
Proposal (let's) → tag = "shall we?" ✓
✗
Let's go for a walk, don't we? [Incorrect]
"don't we?" is wrong for proposals — must use "shall we?" ✗
✓
Let us consider all the possibilities, shall we? [Correct]
Proposal (let us) → tag = "shall we?" ✓
✓
Let us go out to dinner, shall we? [Correct]
Proposal (let us) → tag = "shall we?" ✓
✓
Let's start the meeting, shall we? [Correct]
Proposal → "shall we?" ✓
✓
Let's watch a movie tonight, shall we? [Correct]
Proposal → "shall we?" ✓
💡
Memory Trick:
"Let's/Let us" = you are INVITING someone to do something WITH you = proposal.
Think of it as asking: "Shall we do this together?" — that's exactly your question tag!

Spot "let's" or "let us" at the start → immediately write "shall we?" as the tag.
⭐
Key Exam Rule:
Whenever you see "let's" or "let us" at the start of a sentence → question tag = "shall we?" ALWAYS.
This is one of the most frequently tested question tag concepts in exams!

📋 Rule 75 — Quick Recap

  • 4 types of imperative sentences: Request (please give me…) · Order (close the door) · Advice (respect your parents) · Proposal (let's go…)
  • Proposal ("let's" / "let us") → question tag = "shall we?" ALWAYS
  • "Let's go for a walk, shall we?" ✓ | "don't we?" ✗ · "won't we?" ✗
  • "Let us consider all possibilities, shall we?" ✓
  • "Let us go out to dinner, shall we?" ✓
