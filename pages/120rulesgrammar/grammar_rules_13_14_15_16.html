13

Article "The" with Common Places

Primary Purpose → NO "the"  |  Secondary Purpose → USE "the"

→ The Core Rule

Whenever we use normal, common places like school, college, church, bed, table, hospital, market, prison/jail, court, temple, mosque, university etc., the article "The" is not used before them when these places are used for their primary purpose. But when these places are used for a secondary purpose, the article "The" is used before them.

★ NOTE from the book: It is important to understand the difference between primary purpose and secondary purpose. The primary purpose refers to work for which they are being made, while the secondary purpose refers to any work other than for which they are being made.
✓ PRIMARY Purpose → No "The"

Using the place for the reason it was built / made:

  • School → to study / attend classes
  • Temple → to worship / pray
  • Hospital → to get medical treatment
  • Prison → to serve punishment
  • Church → to worship
  • Bed → to sleep / rest
  • Market → to buy/sell goods
✗ SECONDARY Purpose → Use "The"

Using the place for some OTHER reason (not what it was built for):

  • Temple → to meet someone / attend wedding
  • School → to meet a teacher socially
  • Hospital → to visit someone, not as a patient
  • Church → for a social event
  • Market → just passing by / reference to location
Primary Purpose (what it was made for) → NO article "the"
Secondary Purpose (any other reason) → USE article "the"
→ Which places does this rule apply to?

This rule applies to the following common/normal places:

school college church mosque temple university court jail / prison market hospital table bed

Things which are common / normal in everyday life — not specific named places.

★ All Examples — from book & class

✓

✓

Secondary purpose — he goes for any work other than worship (e.g., to meet someone, to attend a wedding, to visit). → USE "the". [It means he goes there to do any work other than worship] ✓
✓

Secondary purpose — "the temple" here refers to the building as a location/landmark, not as a place of worship. → USE "the". [Secondary purpose] ✓
Church — Worship = Primary
✓

Primary purpose — going every Friday for a proper religious ceremony/worship → NO "the". [Correct] ✓
✗
We go to the church on Fridays.
WRONG — If going every Friday for regular worship (primary purpose), "the" must NOT be used. Using "the" implies a secondary purpose. [Incorrect] ✗
Jail / Prison — Punishment = Primary
✓
The politician was sent to jail.
Primary purpose — jail is used for imprisonment (what it is made for) → NO "the". [Correct] ✓
✗
The politician was sent to the jail.
WRONG — Jail's primary purpose is punishment/imprisonment. Using "the" implies a secondary purpose which doesn't apply here. [Incorrect] ✗
💡
Memory trick — Ask yourself:
"Why is this person going to / using this place?"
→ For the reason it was built (its main job)? → No "the"
→ For any other reason? → Use "the"

Example: Someone going to a temple to steal shoes → secondary purpose → "the temple" ✓ (mam's fun example!)

📋 Rule 13 — Quick Recap

  • Applies to: school, college, church, mosque, temple, university, court, jail/prison, market, hospital, table, bed
  • Primary Purpose (the reason the place was built/made) → NO "the"
  • Secondary Purpose (any other reason than what it was built for) → USE "the"
  • "Children go to school" ✓ (study = primary); "He goes to the temple" ✓ (some other work = secondary)
  • "The politician was sent to jail" ✓ (punishment = primary); "sent to the jail" ✗
  • "We go to church on Fridays" ✓ (worship = primary); "We go to the church on Fridays" ✗

14

"No Sooner…Than" and "Hardly/Scarcely/Barely…When"

Adverbs at the start of a sentence → always followed by a Helping Verb (Inversion)

→ The Core Rule

Whenever adverbs like "No sooner", "Hardly", "Scarcely", "Barely" etc. are used at the beginning of a sentence, they are always followed by a Helping Verb (Law of Inversion applies).

AdverbAlways paired withCorrect Pattern
No sooner "than" (NEVER "when") No sooner + had/did + Subject + V3/V1 + than…
Hardly "when" (NEVER "than") Hardly + had/did + Subject + V3/V1 + when…
Scarcely "when" (NEVER "than") Scarcely + had/did + Subject + V3/V1 + when…
Barely "when" (NEVER "than") Barely + had/did + Subject + V3/V1 + when…
Rarely "when" Rarely + had/did + Subject + … + when…
⚠️
Most common exam mistake:
Students mix up "than" and "when" — or forget to apply inversion (helping verb must come before subject).

"No sooner" → "than"     NOT "when"
"Hardly / Scarcely / Barely" → "when"     NOT "than"
★ NOTE from the book: Note that adverbs "hardly, scarcely and barely" are always followed by "when", while "No sooner" is followed by "than".
★ All Examples from Class & Book
Hardly…When
✓
Hardly had I begun to speak when I was interrupted.
Hardly (Adverb) → had (Helping Verb — inversion!) → I (Subject) → when (correct pairing with "hardly"). ✓ Meaning: I had barely started to speak when I was stopped.
Scarcely…When
✓
Scarcely had we started our meal when the phone rang.
Scarcely → had (inversion) → we (subject) → when ✓
No Sooner…Than
✓
No sooner had I arrived than they all started to argue.
No sooner → had (inversion) → I (subject) → than (correct pairing — NOT "when"). ✓
Incorrect vs Correct — Common Errors
✗
Hardly I entered the class when it began to rain.
WRONG — No inversion! "had" must come before "I" after "Hardly". ✗
✓
Hardly had I entered the class when it began to rain.
CORRECT — Inversion applied: had (HV) before I (Subject). ✓
✗
No sooner had I entered the class when it began to rain.
WRONG — "No sooner" must pair with "than", NOT "when". ✗ [Incorrect]
✓
No sooner had I entered the class than it began to rain.
CORRECT — "than" used with "No sooner". ✓ [Correct]
Today's Homework Question (from class):
"Hardly had I entered the room than it began to rain."
What is the error? → "than" should be "when" — "Hardly" pairs with "when", not "than".
Correct: "Hardly had I entered the room when it began to rain."

📋 Rule 14 — Quick Recap

  • Adverbs at start of sentence → Law of Inversion → Helping Verb comes before Subject
  • "No sooner" → always paired with "than" (NEVER "when")
  • "Hardly / Scarcely / Barely / Rarely" → always paired with "when" (NEVER "than")
  • Pattern: Adverb + Helping Verb (had/did) + Subject + … + (than/when) + …
  • Exam trick: See "No sooner" → look for "than". See "Hardly/Scarcely/Barely" → look for "when".

15

Use of Reflexive Pronoun as the Object

Certain words always need an Object — if no object given, use a Reflexive Pronoun

→ Quick Recap — Subject vs Object

The Object of a sentence is the person or thing that receives the action of the verb. It answers the question "whom?" or "what?"

PartWhat is it?Example
SubjectWho/what the sentence is about. Comes before the verb."I enjoyed the party." → I = subject
VerbThe action"I enjoyed the party."
ObjectWho/what receives the action. Comes after the verb. Must be a Noun or Pronoun."I enjoyed the party." → the party = object
Important: An Object is always a Noun or Pronoun. If a phrase starts with a preposition (e.g., "at the party", "of the opportunity", "from the class"), it is a Prepositional Phrase — it is NOT an object. This is crucial for applying Rule 15.
→ Reflexive Pronouns — What are they?

A Reflexive Pronoun refers back to the subject of the sentence. It is used when the subject and object are the same person.

myselfused with → I
yourselfused with → You (singular)
himselfused with → He
herselfused with → She
itselfused with → It
ourselvesused with → We
yourselvesused with → You (plural)
themselvesused with → They
Rule Certain Words Always Need an Object

There are certain words which are always followed by an object. These words can never stand alone without an object. If no object is given, we use a Reflexive Pronoun as the object.

★ From the book: The object of a sentence is the person or thing that receives the action of the verb. It answers the question "whom" or "what". There are certain words which are always followed by an object: enjoy, avail, absent, pride, adjust, reconcile, acquit, amuse, resign, avenge, exert, apply, adapt, etc. When these words are not followed by an object in the sentence, we use the reflexive pronouns as their object.
enjoy avail absent pride adjust reconcile acquit amuse resign avenge exert apply adapt
Object IS present → No reflexive pronoun needed
No object / only prepositional phrase → Add reflexive pronoun as object
Examples from Book & Class
✗
I enjoyed at the party.
WRONG — "enjoyed" has no object. "at the party" is a prepositional phrase (not an object). Must add reflexive pronoun. ✗
✓
I enjoyed myself at the party.
CORRECT — "myself" acts as the object of "enjoyed". "at the party" is still a prepositional phrase (not the object). ✓
✓
I enjoy the party.
CORRECT — "the party" is already the object of "enjoy". No reflexive pronoun needed. ✓ [Object present]
✓
He enjoyed himself at the party.
"at the party" = prepositional phrase (not an object) → reflexive pronoun "himself" needed as object. [Object = himself] ✓
✗
I availed of the opportunity.
WRONG — "of the opportunity" = prepositional phrase, NOT an object. "avail" needs an object. ✗
✓
I availed myself of the opportunity.
CORRECT — "myself" = reflexive pronoun acting as object. "of the opportunity" = prepositional phrase. ✓ [Object = myself]
✓
I availed the opportunity.
CORRECT — "the opportunity" IS the object. No reflexive pronoun needed. ✓
✓
She availed herself of the opportunity and thus got a promotion.
"herself" = object of "availed". "of the opportunity" = prepositional phrase. ✓ [Object = herself]
✓
They enjoyed the picture last evening.
"the picture" IS the direct object of "enjoyed" → NO reflexive pronoun needed. ✓ [Object = the picture]
✓
The former IPS officer acquitted himself effectively.
"himself" = reflexive pronoun as object of "acquitted". "effectively" = adverb (not an object). ✓
✓
He absented himself from the class.
"himself" = object of "absented". "from the class" = prepositional phrase (not object). ✓
💡
How to check if an object is present:
Ask: "Verb + whom? or what?"
If you get a noun/pronoun directly → that IS the object → no reflexive needed.
If you get a prepositional phrase (at, of, from, in, on + noun) → that is NOT the object → add reflexive pronoun.

Remember: Object = always a Noun or Pronoun. NOT a prepositional phrase.

📋 Rule 15 — Quick Recap

  • Words: enjoy, avail, absent, pride, adjust, reconcile, acquit, amuse, resign, avenge, exert, apply, adapt → always need an object
  • Object IS given in sentence → No reflexive pronoun needed
  • No object / only a prepositional phrase → Add reflexive pronoun (myself/himself/herself…)
  • Prepositional phrase (at, of, from, in… + noun) ≠ Object. Object = noun/pronoun directly after verb
  • "I enjoyed myself at the party" ✓ | "I enjoyed the party" ✓ | "I enjoyed at the party" ✗
  • "I availed myself of the opportunity" ✓ | "I availed of the opportunity" ✗

CORRECT — "the picture" = object already present → no reflexive needed ✓

16

Words NEVER Followed by Reflexive Pronouns

hide, keep, stop, turn, qualify, bathe, move, rest, shave, dress — NO reflexive pronoun

→ The Core Rule

While Rule 15 told us certain words always need an object (and use reflexive pronouns when no object is given), Rule 16 tells us there are certain words that are NEVER followed by a reflexive pronoun — even when these activities are performed on oneself.

★ From the book: There are certain words that are not followed by reflexive pronouns as their object: keep, stop, turn, qualify, bathe, move, rest, hide, etc.

These are all activities that a person typically does to/for themselves — and because they are naturally reflexive (you can't hide someone else when you "hide"), a reflexive pronoun is NOT added.
hide keep stop turn qualify bathe move rest shave dress*
Note about "dress": Modern English has started to use "dress" without a reflexive pronoun. So "He dressed" (not "He dressed himself") is now acceptable. Mark this with a star — both forms may be seen, but the trend is without reflexive.
★ All Examples — from Book & Class

WRONG — "hide" is never followed by a reflexive pronoun. [Incorrect]
✓

CORRECT — No reflexive pronoun after "hid". [Correct]
From the book's explanation: In the given examples, we cannot use the word "himself" as the word "hid" is not followed by a reflexive pronoun.

bathe — Never use reflexive

💡
Memory aid — Natural Actions:
Activities like hide, keep, stop, qualify, bathe, move, rest, shave, dress are things people naturally do for themselves. Adding a reflexive pronoun is redundant and grammatically incorrect in standard English.

📋 Rule 16 — Quick Recap

  • Verbs that never take a reflexive pronoun: hide, keep, stop, turn, qualify, bathe, move, rest, shave, dress.
  • "He hid in the room" ✓ (not "hid himself")
  • "Keep away from bad company" ✓ (not "keep yourself")
  • "He qualified for the post" ✓ (not "qualified himself")
  • "She bathed in the river" ✓ (not "bathed herself")

Rule 13 — Article "The" with Common Places Common Places
Primary Purpose:NO article "the". Using the place for the exact reason it was built (e.g. School to study, Temple to worship, Jail for punishment, Bed to sleep). E.g. "He goes to school every day."
Secondary Purpose:USE article "the". Using the place for any other work or social visit (e.g. visiting a school to meet a teacher, going to a temple to attend a wedding). E.g. "He goes to the temple to meet his friend."
Key Places:school, college, church, mosque, temple, university, court, jail/prison, market, hospital, table, bed.
14
Rule 14 — "No Sooner...Than" & "Hardly...When" Adverb Inversion
Law of Inversion:Adverbs at the start of a sentence require Helping Verb before Subject (Inversion: Adverb + Had/Did + Subject + Verb...).
No Sooner pairing:Always paired with "than" (never "when"). E.g. "No sooner had I arrived than they started arguing."
Hardly/Scarcely/Barely:Always paired with "when" (never "than"). E.g. "Hardly had I entered when it began to rain."
15
Rule 15 — Reflexive Pronoun as the Object Reflexive Objects
Rule:Certain transitive verbs must always be followed by an object. If no direct noun/pronoun object is present (e.g. only a prepositional phrase like "at the party"), a reflexive pronoun must be added.
Verbs affected:enjoy, avail, absent, pride, adjust, reconcile, acquit, amuse, resign, avenge, exert, apply, adapt.
Examples:"I enjoyed myself at the party" (no object, so add "myself") vs "I enjoyed the party" (object present). "I availed myself of the opportunity."
16
Rule 16 — Words NEVER Followed by Reflexive Pronouns Exempt Verbs
Rule:Certain verbs represent actions naturally performed on oneself and must never be followed by a reflexive pronoun.
Verbs affected:hide, keep, stop, turn, qualify, bathe, move, rest, shave, dress.
Examples:"He hid in the room" (not "hid himself"). "You should keep away from bad company" (not "keep yourself"). "She bathed in the river" (not "bathed herself").
