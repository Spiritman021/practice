📘 Practice Hub India — README.md

🎯 परियोजना का उद्देश्य (Project Overview)

Practice Hub India एक सरकारी परीक्षा अभ्यर्थियों के लिए डिज़ाइन किया गया डिजिटल वर्कस्पेस है। यह पोर्टल SSC CGL, Banking (IBPS PO/SO/Clerk), Railway (RRB NTPC/Group D) और अन्य सरकारी परीक्षाओं की तैयारी में सहायता प्रदान करता है।

📂 फ़ाइल संरचना (File Structure)

  • index.html — मुख्य पोर्टल (Main Portal)
  • pages/vocab/ — दैनिक शब्दावली (Daily Vocabulary)
  • pages/grammar/ — व्याकरण नियम (Grammar Rules)
  • pages/computer/ — कंप्यूटर विज्ञान (Computer Science)
  • pages/tools/ — गणित उपकरण (Math Tools)
  • pages/120rulesgrammar/ — 120 व्याकरण नियम
  • pages/grammar-granth/ — Grammar Granth Notes
  • pyqs/ — पिछले वर्ष के प्रश्न (PYQ Analysis)
  • pyqs/ibps-so-rc-tone-bank.html — ✨ Tone of the Passage Guide (नई!)

🆕 नवीनतम अपडेट (Latest Updates)

  • ✨ Tone of the Passage Study Guide जोड़ी गई — 21+ tones with Hindi hints, synonyms, examples and exam frequency
  • 📚 Grammar Granth Day 1, 2, 3 notes added
  • 📋 IBPS SO PYQ Analysis updated
  • 🔢 Consecutive Products tool added

⚙ तकनीकी जानकारी (Tech Stack)

  • HTML5 + CSS3 + Vanilla JavaScript
  • Google Fonts: Outfit
  • PWA Support: manifest.json + sw.js
  • Hosted: GitHub Pages (spiritman021.github.io/practice)
